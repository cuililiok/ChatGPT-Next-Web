# AI 估值诊断：案例拆解
## 本文件：三类典型公司的完整诊断示范

---

## 案例 A：云算力基础设施公司（"卖铲子"象限）
*以 Nvidia 为参照原型*

### 维度一：价值链位置
- 象限：**卖铲子**
- AI 是核心生产要素，公司直接销售算力（GPU）
- 价值链判断：短期内价值向这里聚集，长期取决于是否有替代架构出现（定制 ASIC、量子计算）
- **修正方向：** 利润率目标高于行业均值，但需设过渡期——一旦竞争者推出替代架构，利润率会迅速收敛

### 维度二：数据护城河
- 评分：3/10（硬件公司数据护城河弱，主要护城河来自 CUDA 软件生态）
- **修正：** 不做数据资产单独估值。CUDA 生态锁定可在"超额 ROIC 持续时间"假设里体现（延长至 7–10 年而非行业均值 5 年）

### 维度三：算力成本曲线
- 适用？是（公司销售算力，其产品价格受算力通缩影响）
- 特殊性：卖铲子公司受益于**需求扩张**大于受损于**单价下降**（只要 AI 训练规模持续扩大）
- **修正：** 在情景分析里设"算力需求增速放缓"为悲观情景触发器，而不是算力单价下降

### 维度四：监管期权
- 主要风险：出口管制（美国对中国的芯片出口限制）
- TAM 调整：
  - 严格管制情景（P=35%）：TAM × 0.65（中国市场占 ~35% 的潜在需求）
  - 现状情景（P=45%）：TAM × 1.0
  - 放松情景（P=20%）：TAM × 1.15
  - TAM_adjusted = 0.35×0.65 + 0.45×1.0 + 0.20×1.15 = 0.90 × TAM_base
- **修正：** 长期收入上限打九折

### 维度五：人力资本折旧
- 不适用（硬件公司，AI 替代人力不是核心变量）

### 参数修正单
```
目标利润率（终值）：行业均值 +5 ppt（CUDA 生态锁定）
超额 ROIC 持续期：10年（而非标准5年）
TAM 上限：× 0.90（出口管制调整）
情景：增加"算力需求放缓"为独立悲观触发器
```

## 案例 B：AI 大模型公司（"新平台"象限）
*以封闭模型 API 公司为参照原型*

### 维度一：价值链位置
- 象限：**新平台**（AI 是核心要素 + 受益方）
- 价值链判断：基础模型层是否会成为平台垄断，取决于：
  1. 开源模型能否追上闭源能力（目前差距在缩小）
  2. 应用层是否会向自研模型迁移（风险：成为商品）
  3. 是否有数据 + 推理 + 应用的垂直整合（Apple 式防守）
- **关键判断：** 这是双峰分布公司，需要用双峰框架，不用标准三情景

### 维度二：数据护城河
- 评分：8/10（大规模专有训练数据，RLHF 数据具有网络效应——用户越多，反馈越好）
- **修正：** 建议单独估值数据资产
  - 超额利润来源：数据驱动的模型性能领先 → 定价溢价 15–20%
  - 数据资产估值：超额利润率 × 收入 / WACC ≈ 单独加值

### 维度三：算力成本曲线
- 适用（核心成本是训练+推理算力）
- **两阶段 Sales/Capital：**
  - 训练期（Y1–Y3）：Sales/Capital = 0.4–0.7（大量算力采购，收入滞后）
  - 推理商业化期（Y4–Y10）：Sales/Capital = 2.5–5.0（边际成本趋零）
- 算力通缩对目标利润率：向上修正 5–8 ppt（推理成本每年下降 30–40%）

### 维度四：监管期权
```
欧盟（高风险系统分类）：P(严格)=40%, TAM×0.70
美国（联邦宽松+州分裂）：P(友好)=50%, TAM×1.10
中国（备案制）：P(被阻)=60%, TAM×0.20（外国模型极难进入）
综合 TAM_adjusted ≈ 0.85 × TAM_base（乐观估计）
```

### 维度五：人力资本折旧
- 方向 B（效率红利端）：公司内部人力成本下降（用 AI 做 AI）
- 但警惕：竞争者有相同效率提升，红利会被竞争压平
- **修正：** 短期利润率+3 ppt，终值利润率不调整

### 双峰分布处理
```
V = 0.35 × V(平台垄断) + 0.65 × V(商品化/边缘化)

V(平台垄断)：利润率45%, ROIC持续高于WACC 15年, 高市占
V(商品化)：利润率8-12%, ROIC≈WACC, 接近EPV

加权估值大概率远低于纯乐观DCF，也远高于纯悲观DCF
```

### 参数修正单
```
Sales/Capital（Y1–3）：0.5（低于行业均值）
Sales/Capital（Y4–10）：3.5（高于行业均值）
目标利润率（短期）：行业均值+3 ppt（算力通缩+效率红利）
目标利润率（终值）：保持行业均值（竞争压平）
TAM 上限：×0.85（监管调整）
情景结构：双峰（垄断35%/商品化65%）而非标准三情景
数据资产独立加值：按超额利润率法计算
```

## 案例 C：被 AI 冲击的知识服务公司（"被替代者"象限）
*以卖方研究/基础法律服务公司为参照原型*

### 维度一：价值链位置
- 象限：**被替代者**
- 核心产品是"人工专业判断"，AI 可以大规模复制这类输出
- 关键问题：替代发生在哪个层级？
  - 执行层（格式化报告、标准合同）：替代快（2–4年）
  - 研判层（战略建议、复杂谈判）：替代慢（>7年）
- **修正：** 区分收入中的执行层比例，对执行层收入做定价权折价

### 维度二：数据护城河
- 评分：2/10（行业研究数据大多可外购，无专有网络效应）
- **修正：** 不做单独估值，但在叙事里注明"无数据护城河加速竞争同质化"

### 维度三：算力成本曲线
- 不适用（公司不持有算力）

### 维度四：监管期权
- 金融研究的监管（MiFID II 等）已经存在，AI 进入不改变监管框架
- 主要监管风险：AI 生成研究的免责声明要求、分析师责任界定
- **修正：** 监管期权影响有限，不做 TAM 调整，但合规成本+0.5 ppt 运营成本

### 维度五：人力资本折旧
- 方向 A（被替代端）
- 卖方研究执行层替代速度：中快（3–5年显著）
- 估算：执行层占总收入约 40%，定价权流失导致该部分利润率下降 8 ppt
- 综合影响：目标利润率下调 3.2 ppt（40% × 8 ppt）

### 参数修正单
```
目标利润率（Y3–5）：行业均值 -3 ppt（执行层定价权流失）
目标利润率（终值）：行业均值 -5 ppt（长期结构性压力）
超额 ROIC 持续期：缩短至 3年（而非标准5年）
情景：增加"AI 替代加速"悲观情景（权重 40%），
      并考虑清算价值作为下限锚点
是否讨论清算价值：是（若利润率持续低于 WACC，需评估清算vs继续经营）
```
