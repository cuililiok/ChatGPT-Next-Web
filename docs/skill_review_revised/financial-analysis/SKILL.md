# 上市公司专业财务分析 Skill

## 核心理念

> "会计是商业的语言，但财报只说它想说的话。" —— 改编自巴菲特

分析分三个层次，顺序不可颠倒：
1. **审计师视角**：财报是否真实可信？有无造假或操纵迹象？
2. **分析师视角**：盈利质量如何？增长是否可持续？
3. **投资大师视角**：这是不是一门好生意？值多少钱？

**分析顺序永远是**：先验真伪 → 再看质量 → 最后谈价值

## 第一层：财务真实性核查（审计师视角）

> 详细造假识别清单见 `references/fraud-detection.md`

### 1.1 三表勾稽关系验证

**利润表 → 现金流量表（最核心）**
- **现金含金量 = 经营活动现金净流量 / 净利润**
  - 优质：长期 > 1.0 ✅ | 警示：< 0.8 🟡 | 危险：< 0 🚨
- 净利润高但经营CF持续为负 = 利润注水最强信号

**资产负债表内部一致性**
- 净利润 + 期初留存 = 期末留存（±分红），若不等则有重大问题
- 货币资金高 + 仍大量借债 = 资金真实性存疑（康美药业模式）

**现金流量表四象限判断**

| 经营CF | 投资CF | 筹资CF | 判断 |
|--------|--------|--------|------|
| ＋ | － | － | ✅ 成熟优质，自我造血还债 |
| ＋ | － | ＋ | 🔵 扩张期，关注杠杆水平 |
| － | － | ＋ | 🔴 烧钱期/危机，需深究 |
| － | ＋ | ＋ | 🚨 高危：卖资产+借钱维持运营 |

### 1.2 十大财务造假红旗

| # | 信号 | 快速核查方法 |
|---|------|------------|
| 1 | 收入增速 >> 现金流增速 | 现金含金量连续2年 < 0.7 |
| 2 | 应收账款增速 >> 收入增速 | 周转天数同比拉长 > 20天 |
| 3 | 存货积压 | 存货周转天数行业对比 |
| 4 | 账上现金高但持续融资 | 货币资金利息收入是否匹配 |
| 5 | 毛利率显著高于同行 | 可能是护城河，也可能是造假 |
| 6 | 非标审计意见 | 直接阅读审计报告意见段 |
| 7 | 频繁更换审计机构 | 四大→小所 = 重大警示 |
| 8 | 大额商誉长期不减值 | 被收购方当前实际盈利情况 |
| 9 | 管理层大量减持 | 尤其在业绩高峰期 |
| 10 | 关联交易复杂 | 关联应收/关联销售占比 |

## 第二层：盈利质量分析（财务分析师视角）

> 各行业专项指标见 `references/industry-specific.md`
> 完整公式见 `references/ratio-formulas.md`

### 2.1 收入质量

- **量价拆分**：收入增长来自量增还是提价？（提价含金量更高）
- **有机 vs 外延**：并购贡献剔除后，内生增速是多少？
- **非经常性损益剔除**：扣非净利润 / 净利润，长期应 > 0.9
- **合同负债趋势**：增加 = 客户提前打款 = 需求旺盛的强信号

### 2.2 毛利率与费用分析

```
毛利率：> 40% 通常意味着定价权，需持续跟踪趋势
EBITDA利润率：剔除折旧影响，跨行业比较更公平
三费率（销售+管理+财务）：费用率下降 = 规模效应显现
研发资本化率：< 30% 为保守，> 50% 利润可能虚高
```

### 2.3 杜邦分析（ROE 三因素分解）

**ROE = 净利润率 × 资产周转率 × 财务杠杆**

| ROE 驱动因素 | 代表类型 | 可持续性 |
|-------------|---------|---------|
| 高净利率 | 品牌/技术溢价（茅台、苹果）| ✅ 高 |
| 高周转率 | 渠道/零售效率（沃尔玛、海天）| 🔵 中 |
| 高杠杆 | 金融/地产 | 🔴 低，周期风险大 |

**巴菲特标准**：ROE > 15% 连续10年，且杠杆不高

### 2.4 营运效率（CCC 现金转换周期）

```
CCC = 应收账款周转天数 + 存货周转天数 - 应付账款周转天数

CCC < 0 = 极佳（占用上下游资金，如亚马逊、茅台）
CCC 0~60天 = 优质
CCC > 90天 = 需关注营运资金压力
```

## 第三层：资产负债表健康度

### 3.1 资产质量排序（含金量从高到低）

1. 货币资金（需验证真实性）
2. 应收账款（账龄结构 + 坏账率）
3. 存货（跌价准备充分性）
4. 固定资产（折旧政策合理性）
5. 无形资产/商誉（主观估值，减值风险高）

**商誉风险红线**：商誉 / 净资产 > 30% 需专项分析

### 3.2 偿债能力

| 指标 | 健康值 | 警示线 | 危险线 |
|------|--------|--------|--------|
| 流动比率 | > 2.0 | < 1.5 | < 1.0 |
| 速动比率 | > 1.0 | < 0.8 | < 0.5 |
| 资产负债率 | < 50% | > 65% | > 80% |
| 净负债率 | < 50% | > 100% | > 200% |
| 利息保障倍数 | > 5x | < 3x | < 1.5x |

## 第四层：成长性与资本配置（投资大师视角）

### 4.1 自由现金流（FCF）——最核心指标

```
FCF = 经营活动现金净流量 - 资本支出

FCF Yield = FCF / 市值（> 5% 通常有吸引力）
FCF / 净利润 > 1.0 = 优质（现金转化效率高）
```

巴菲特将"所有者盈余"（≈FCF）视为企业真实价值创造的唯一标准。

### 4.2 资本配置优先级（检验管理层智慧）

优秀管理层的资本配置顺序：
1. 高回报率的内部再投资（ROIC > WACC）
2. 无好项目时 → 分红或回购（把钱还给股东）
3. 并购（警惕溢价过高的"帝国建造"行为）

**资本轻重判断**：
- Capex / 收入 < 5%：轻资产，现金流充裕 ✅
- Capex / 折旧 < 1.0：维持性支出，无增长投入 🟡
- Capex / 折旧 > 2.0：大规模扩张期，关注回报率 🔵

### 4.3 股东回报质量

- 分红率 > 30%（成熟期优质公司）
- 回购是否在合理估值区间执行？（高价回购 = 摧毁价值）
- SBC / 净利润 < 10%（股权激励不过度稀释股东）

## 第五层：估值分析

> 详细估值方法与DCF模板见 `references/ratio-formulas.md`

### 5.1 估值方法选择

| 公司类型 | 首选方法 | 辅助方法 |
|---------|---------|---------|
| 成熟盈利型 | PE、EV/EBITDA | DCF、PEG |
| 重资产/周期 | PB、EV/EBITDA | 底部PE |
| 高增长/亏损 | EV/Revenue、PS | DCF终局视角 |
| 银行/保险 | PB、内含价值 | PE、DDM |

### 5.2 安全边际原则（格雷厄姆）

```
目标买入价 = DCF内在价值 × 60%~70%（预留30%~40%安全边际）

估值区间判断：
PE历史分位 < 30% = 历史低估区间
PE历史分位 > 70% = 历史高估区间

PEG < 1：增速支撑估值，通常低估
PEG > 2：需要极高确定性才能支撑
```

## 第六层：公司治理与管理层

> 芒格：投资一家你不信任其管理层的公司，就像与骗子做生意。

### 6.1 管理层诚信核查清单

**积极信号** ✅
- 年报直面失败，量化复盘
- 薪酬与长期股价/ROE挂钩
- 股东信息息量大、语言真诚
- 回购在低估值区间执行

**危险信号** 🚨
- 频繁更换CFO/审计委员会
- 业绩高峰期大规模减持
- 关联交易复杂，信批不透明
- 承诺业绩连续不达标但仍大量激励

### 6.2 股权结构风险

- 股权质押比例 > 50% = 股价下行触发强平的系统风险
- 实控人持股比例过低 = 控制权不稳定
- 外资持仓变化趋势（长期配置的机构资金信号）
