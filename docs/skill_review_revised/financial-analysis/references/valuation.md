# 估值方法详解

## 一、绝对估值法：DCF

### 1.1 WACC 计算
WACC = Ke × (E/V) + Kd × (1-t) × (D/V)

- Ke（权益资本成本）= 无风险利率 + β × 市场风险溢价
  - 中国：无风险利率参考10年期国债（约2.5~3%）
  - 美国：无风险利率参考10年期美债
  - 市场风险溢价：中国约6~8%，美国约5~6%
- Kd（债务资本成本）= 实际借款利率 × (1 - 税率)
- 典型WACC范围：A股优质公司 8~12%，美股 7~10%

### 1.2 终值计算
永续增长模型：终值 = FCFn × (1+g) / (WACC - g)
- g（永续增长率）：一般不超过GDP名义增速（约3~5%）
- 终值通常占总价值的60~80%，对假设非常敏感

### 1.3 DCF 敏感性分析模板
构建 WACC × 永续增长率 的二维敏感性表：
- WACC 范围：基准 ± 2%（每格0.5%）
- g 范围：1% ~ 5%（每格1%）
- 取中间 9 格的价值范围作为合理估值区间

## 二、相对估值法详解

### 2.1 PE（市盈率）
PE = 市值 / 净利润 = 股价 / EPS

适用场景：稳定盈利的成熟公司
注意事项：
- 用"正常化利润"而非当期利润（剔除非经常性损益）
- 周期底部PE高（利润低），周期顶部PE低（利润高）——周期股反着用

历史参考分位（A股）：
- 消费白马：15~30x 为合理区间
- 科技成长：25~50x
- 金融地产：5~12x

### 2.2 PB（市净率）
PB = 市值 / 净资产

适用场景：重资产行业（银行、地产、钢铁）、资产价值相对稳定的公司
注意事项：
- 净资产质量至关重要（商誉/无形资产占比高时PB失真）
- PB < 1 不等于便宜（净资产可能持续消耗）

### 2.3 EV/EBITDA
EV = 市值 + 有息负债 - 货币资金
EBITDA = EBIT + 折旧 + 摊销

优势：
- 剔除资本结构差异，适合跨国/跨行业比较
- 折旧摊销多的重资产行业（地产、矿业、通信）比PE更合适

参考倍数：
- 制造业：6~10x
- 消费/品牌：10~20x
- 科技：15~30x

### 2.4 PS（市销率）
PS = 市值 / 营业收入

适用场景：亏损成长公司（SaaS、互联网早期）
注意事项：
- 需结合毛利率判断（高PS必须有高毛利支撑）
- 毛利率 > 70% 的SaaS公司：合理PS 5~15x
- 毛利率 < 30% 的公司：PS > 3x 通常偏贵

### 2.5 DDM（股息折现模型）
适用场景：高股息稳定分红公司（公用事业、银行、保险）
P = D1 / (Ke - g)

股息率 = 每股股息 / 股价 > 4% 通常具备配置价值（低利率环境）

## 三、特殊场景估值

### 3.1 银行股估值
- 核心指标：PB（账面价值） + ROE
- 格雷厄姆公式：合理PB = ROE / Ke
  例：ROE=12%，Ke=10%，则合理PB = 1.2x
- 不良贷款率 < 1.5% 为健康；拨备覆盖率 > 150% 为安全

### 3.2 保险股估值
- 内含价值（EV）：现有保单未来利润的精算现值
- P/EV < 1x 通常低估；1~1.5x 合理；> 2x 偏贵
- 新业务价值（NBV）增速是核心成长指标

### 3.3 地产股估值（NAV法）
NAV = 土地储备价值 + 在建项目价值 + 已完工项目价值 - 净负债
折扣率：优质地产公司通常以NAV 7~9折交易

### 3.4 矿业资源股
EV / 储量：每吨/盎司资源的市场定价
与同类资产对比（同品位、同区位）

## 四、估值综合校验

### "三法合一"校验原则
对同一家公司同时使用3种方法估值：
1. DCF（绝对价值锚）
2. 相对估值（市场共识）
3. 历史估值分位（时间维度）

三种方法结果收敛 → 估值可信度高
三种方法结果发散 → 需找出假设差异，谨慎决策

### 逆向思维：市场隐含假设
当前股价隐含了什么增长率？
反推：若当前PE=30x，无风险利率3%，则市场隐含增长率约= 1/30 × (1+g)/(WACC-g)
判断这个隐含增长率是否合理，是买卖决策的本质。
