# 02 盈利能力深度解剖

## 毛利率分析体系

### 毛利率的行业基准
```
超高毛利率（> 60%）：软件SaaS、品牌奢侈品、创新药、高端白酒
高毛利率（40%~60%）：消费品龙头、医疗器械、互联网平台
中等毛利率（20%~40%）：制造业龙头、食品饮料、连锁零售
低毛利率（< 20%）：贸易流通、原材料、重化工、建筑
```

### 毛利率变化的深层解读
```
毛利率上升原因分析：
✅ 产品提价成功（定价权）→ 高质量提升
✅ 规模效应降低单位成本 → 高质量提升
✅ 产品结构升级（高毛利产品占比提升）→ 高质量提升
⚠️ 原材料成本下降（不可持续）→ 中性
❌ 会计处理变更（费用资本化）→ 虚假提升
❌ 关联交易定价操控 → 虚假提升

毛利率下降原因分析：
⚠️ 原材料涨价（周期性压力）→ 关注传导能力
❌ 竞争加剧被迫降价 → 护城河侵蚀
❌ 产品老化/结构恶化 → 长期趋势风险
```

## ROE杜邦体系深度应用

### 五因素杜邦分解（更精细）
```
ROE = 税后利润率 × 利息负担率 × EBIT利润率 × 资产周转率 × 权益乘数

= (净利润/税前利润) × (税前利润/EBIT) × (EBIT/营收) × (营收/总资产) × (总资产/净资产)

解读层次：
- 税后利润率：反映税负影响
- 利息负担率：反映财务杠杆成本
- EBIT利润率：反映经营盈利能力（核心）
- 资产周转率：反映运营效率
- 权益乘数：反映财务杠杆
```

### 不同ROE来源的质量排序
```
排名一：高净利率驱动（品牌/技术护城河）
→ 代表企业：茅台、海天味业、腾讯
→ 护城河最强，最可持续

排名二：高周转率驱动（运营效率护城河）
→ 代表企业：沃尔玛、顺丰、海底捞
→ 护城河较强，依赖精细管理

排名三：高杠杆驱动（放大ROE）
→ 代表企业：地产、银行（行业特性）
→ 需关注杠杆可持续性和利率敏感性

警惕：ROE高但ROIC低
→ 说明杠杆贡献了大部分ROE，实际经营效率一般
```

## ROIC深度应用

### ROIC计算详解
```
NOPAT（息前税后利润）= EBIT × (1 - 税率)
     = 营业利润 - 所得税调整

投入资本 = 运营性资产 - 运营性负债
= (流动资产 - 超额现金 - 非运营资产) - (流动负债 - 短期借款)
+ 固定资产净值 + 商誉 + 无形资产

或简化：
投入资本 = 总资产 - 超额现金 - 无息流动负债
```

### ROIC与WACC的比较
```
WACC计算（简化）：
WACC = 股权成本 × 股权占比 + 债务成本 × (1-税率) × 债务占比

股权成本（CAPM）= 无风险利率 + Beta × 股权风险溢价

判断标准：
ROIC - WACC > 5%  → 显著价值创造，给予高估值溢价
ROIC - WACC ≈ 0   → 保本经营，给予行业平均估值
ROIC - WACC < 0   → 价值销毁，应给予折价估值（PB < 1）
```

### ROIC趋势分析
```
理想模式：ROIC高且稳定（护城河存在）
成长模式：ROIC逐年提升（规模效应显现）
衰退预警：ROIC持续下滑（竞争加剧/护城河侵蚀）
均值回归：ROIC异常高时，关注均值回归时间表
```

## 费用效率分析

### 三费率标准参考
```
销售费用率：
品牌消费品：15%~30%（广告+渠道）
工业品B2B：5%~15%
软件SaaS：25%~40%（获客成本高）

管理费用率：
成熟大型企业：5%~10%
中小成长企业：10%~20%
过高警惕：>25%（管理效率低下或费用藏利润）

研发费用率：
互联网/科技：10%~20%
医疗器械/创新药：15%~30%
消费品：1%~5%（品牌驱动型）
制造业：3%~8%
```

### 费用趋势的战略信号
```
销售费用率上升：
→ 市场竞争加剧（烧钱抢份额）→ 关注转化效率

研发费用率提升：
→ 积极信号（未来增长储备）→ 关注研发成果转化

管理费用率下降：
→ 规模效应体现 → 积极信号

人均产出趋势（营收/员工数、利润/员工数）：
→ 持续提升 = 效率改善 → 强烈正向信号
→ 持续下降 = 效率恶化 → 预警
```

## 行业横向对比方法

### 可比公司选择原则
```
① 相同细分行业（颗粒度一致）
② 相近商业模式（B2B vs B2C需区分）
③ 相近规模阶段（初创期 vs 成熟期不宜对比）
④ 会计政策相近（折旧政策/收入确认方式）
```

### 相对盈利能力评分
```
毛利率：高于行业均值 > 5pct → +2分
净利率：高于行业均值 > 3pct → +2分
ROE：高于行业均值 > 3pct，且持续3年 → +3分
ROIC：持续高于WACC → +3分
总分 > 8分 → 盈利能力优秀
```
