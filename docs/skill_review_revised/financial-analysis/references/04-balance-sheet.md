# 04 资产负债表健康度分析

## 资产质量全面评估

### 应收账款深度分析

```
账龄结构标准（一般行业）：
1年以内：> 80%（正常）
1~2年：  < 10%（警惕上升趋势）
2年以上：< 5%（若超过此比例，需深究坏账风险）

坏账准备计提充分性检验：
① 与同行计提比例对比（政策是否过于宽松？）
② 看账龄分布与计提比例的匹配性
③ 关注"5年以上应收账款"（几乎=损失）

DSO（应收账款周转天数）分析：
DSO = 应收账款 / 日均营收
行业对比 + 历史趋势：
  DSO持续增加 → 回款恶化或收入水分
  DSO突然大幅下降 → 排除是否通过票据贴现
```

### 存货质量分析

```
存货周转天数（DIO）：
DIO = 存货 / 日均营业成本

行业参考：
零售行业：20~40天（周转快）
制造业：  60~120天（生产周期）
重工业：  120~180天（正常）
地产：    数年（特殊行业）

存货风险信号：
① DIO持续上升（积压风险）
② 存货金额增速 >> 营收增速（备货异常）
③ 存货跌价准备计提突然大幅增加（历史数据掩盖问题）
④ 成品占存货比例持续上升（产品卖不动）
⑤ 原材料大幅减少而在产品/成品上升（生产异常）

存货减值充分性：
对照市场价格判断存货是否存在减值迹象
周期性行业（钢铁/化工）在价格下行期需特别关注
```

### 固定资产与在建工程

```
在建工程转固分析：
长期滞留在建工程（超过正常工期）→ 预警
  可能原因：
  ① 项目存在质量问题
  ② 虚增资产（将费用资本化）
  ③ 项目实际已暂停

折旧政策合理性：
固定资产使用年限是否与实际相符？
  偏长 → 少提折旧，虚增利润
  偏短 → 保守稳健（正面）

资产减值充分性：
生产设备是否持续不达产 → 需测试减值
关闭的生产线是否已计提减值
```

### 商誉与无形资产专项分析

```
商誉减值测试关键问题：
① 被收购业务的实际业绩 vs 承诺业绩
② 收购时使用的折现率假设是否合理
③ 永续增长率假设是否过于乐观

商誉暴雷风险评估：
高风险：商誉/净资产 > 50%
        被收购业务持续亏损
        收购时PE > 30倍
        并购后管理层离职

无形资产分析：
技术类无形资产摊销年限（行业惯例5~10年）
客户关系类无形资产（核心是护城河评估）
品牌类无形资产（自创品牌通常无法确认）
```

### 货币资金真实性验证

```
"货币资金陷阱"识别（A股特有风险）：
① 账上大量现金 + 大量有息借款并存
   → 可能：子公司资金被限制；资金被大股东占用
   
② 货币资金 >> 实际经营需要
   但同时：大额分红很少；资本支出合理
   → 深查是否存在虚假银行存款

③ 查证方法：
   - 看期末银行存款函证（审计程序）
   - 看是否存在"受限资金"披露
   - 对比现金余额与财务费用（大量存款却高额利息支出→蹊跷）

受限资金类型：
质押保证金、诉讼保全冻结、特定用途资金
→ 从货币资金中剔除，才是真实可动用资金
```

## 负债结构深度分析

### 债务安全性矩阵

```
短期偿债能力：
流动比率 = 流动资产 / 流动负债
  > 2.0：充裕（可能资产利用效率偏低）
  1.5~2.0：健康
  1.0~1.5：偏紧，需关注
  < 1.0：存在短期流动性风险

速动比率 = (流动资产 - 存货) / 流动负债
  > 1.0：健康
  < 0.8：偏紧

现金比率 = 货币资金 / 流动负债
  > 0.5：充裕
  < 0.2：偏紧（但需结合融资能力）
```

### 中长期偿债能力

```
有息负债/EBITDA（最常用）：
< 1倍：极低杠杆，财务非常稳健
1~3倍：健康
3~5倍：偏高，需关注
> 5倍：高风险（银行评级临界值）

净负债/EBITDA：
= (有息负债 - 现金) / EBITDA
负值：净现金状态（优质）
< 2倍：健康
> 4倍：高度警惕

利息保障倍数（EBIT/利息费用）：
> 10倍：极强（几乎没有偿债压力）
5~10倍：健康
3~5倍：一般
< 3倍：需密切关注
< 1倍：利息都付不起，极度危险
```

### 债务期限结构与再融资风险

```
短期/长期有息负债比例：
短期占比 > 60% → 再融资压力大，关注信用状况

到期债务时间表分析：
查看未来1~3年的到期债务规模
对比年均FCF判断偿债来源

隐性债务识别：
① 经营性租赁（IFRS 16后已大部分入表）
② 对外担保（可能带来或有负债）
③ 表外融资（特殊目的实体）
④ 供应链金融（核心企业保理应收款）
```

## 股东权益质量分析

```
净资产含金量评估：
净资产 = 股本 + 资本公积 + 留存收益 + 其他综合收益

警惕项：
① 其他综合收益大额负数（汇兑损失/金融资产公允价值损失）
② 留存收益长期未增长（持续亏损或大额分红）
③ 资本公积 >> 实缴资本（多次融资稀释）

股权结构稳定性：
控股股东持股比例趋势：持续减持 → 重要预警信号
股权质押率 > 70% → 控股权面临风险
实控人与上市公司资产独立性（避免同业竞争）
```
