# 06 估值方法体系

## 估值核心原则

> "价格是你付出的，价值是你得到的。" — 巴菲特

**四大原则：**
1. 估值是艺术，不是精确科学 → 给出区间，而非单一数字
2. 永远要求安全边际 → 内在价值的70%~80%买入
3. 用多种方法交叉验证 → 单一方法结论不可靠
4. 不确定性越大，安全边际要求越高

## PE估值体系（最常用）

### 正常化净利润的计算
```
剔除项（不可持续收益）：
- 一次性政府补贴
- 资产处置损益（非主业）
- 公允价值变动损益
- 非经营性投资收益

加回项（被低估的经营能力）：
- 当期大额一次性损失（非系统性）
- 新业务初期亏损摊销（主业已验证）

正常化净利润 = 净利润 ± 以上调整项
```

### PE历史分位估值
```
步骤：
① 计算历史5~10年PE分位（25%/50%/75%/90%）
② 判断当前PE处于哪个分位区间
③ 结合当前增速、行业地位判断合理分位

分位区间参考：
< 25%分位：历史低估区（高度关注）
25%~50%分位：合理偏低（适合建仓）
50%~75%分位：合理（持有）
> 75%分位：历史偏高（谨慎追高）
> 90%分位：历史高估（考虑减仓）

注意：不同增速阶段对应合理PE不同
高增速期（> 30%）：高PE合理
低增速期（< 10%）：低PE才合理
```

### PEG估值
```
PEG = PE / 净利润增速（用未来3年CAGR）

格雷厄姆/彼得·林奇标准：
PEG < 0.5：严重低估（罕见）
PEG = 0.5~1.0：低估区间（值得关注）
PEG = 1.0~1.5：合理区间
PEG > 1.5：偏贵
PEG > 2.0：高估（需要非常确定性的高增速支撑）

PEG的局限性：
- 仅适用于成长型公司（增速 > 15%）
- 增速预测误差会导致PEG失真
- 周期类、金融类不适用
```

## DCF估值（内在价值的黄金标准）

### 三段式DCF模型

```
第一阶段：明确预测期（1~5年）
  - 基于公司指引、行业分析、历史趋势
  - 逐年预测：营收 → 利润率 → NOPAT → CapEx → FCF
  
第二阶段：过渡期（6~10年）
  - 增速从第一阶段向永续增长率平滑过渡
  - 反映竞争均值回归
  
第三阶段：永续期（10年后）
  - 永续增长率g：通常2%~3%（不超过GDP长期增速）
  - 终值 = FCF₁₀ × (1+g) / (WACC - g)
  - 终值在总价值中占比通常60%~75%（敏感性高）
```

### WACC计算实务
```
WACC = Ke × E/(D+E) + Kd × (1-T) × D/(D+E)

股权成本（Ke）= 无风险利率 + Beta × 股权风险溢价
  中国A股参考：
  无风险利率：10年期国债收益率（约2.5%~3.5%）
  股权风险溢价（ERP）：约5%~6%
  Beta：参考Bloomberg/Wind，行业调整

债务成本（Kd）：
  = 加权平均借款利率（年报附注中的有息负债）

实务简化：
高质量消费品：WACC约8%~10%
科技成长公司：WACC约10%~12%
周期/高杠杆：WACC约12%~15%
```

### DCF敏感性分析（必做）
```
对WACC和g做双变量敏感性表：

         永续增长率g
WACC  |  1%   2%   3%   4%
  8%  | 80元  90元 105元 130元
  9%  | 70元  78元  90元 108元
 10%  | 62元  68元  78元  92元
 11%  | 55元  60元  68元  78元

→ 分析内在价值的稳健区间
→ 安全边际 = 中性假设下内在价值 × 0.7~0.8
```

## PB估值（资产类公司）

### PB适用场景
```
最适合：金融类（银行/保险/证券）、地产、重资产制造
不适合：轻资产/品牌驱动/软件类（账面净资产意义不大）
```

### ROE-PB模型（最重要的PB估值框架）
```
合理PB = ROE / WACC × (1 - g/ROE)
≈ ROE / Ke（简化）

例：
ROE = 20%，股权成本Ke = 10%
→ 合理PB ≈ 20/10 = 2倍

实践经验：
ROE 10%：合理PB约1倍
ROE 15%：合理PB约1.5~2倍
ROE 20%：合理PB约2~3倍
ROE 30%（茅台级）：合理PB约5~8倍

PB < 1：净资产折价，要么低估，要么ROE长期 < WACC
```

## EV/EBITDA估值

### 适用场景
```
优于PE的场景：
① 比较不同资本结构的公司（消除杠杆差异）
② 高折旧/摊销行业（如电信、制造）
③ 跨国公司比较（消除税率差异）
④ 并购定价（收购方视角）
```

### EV/EBITDA参考区间
```
EV = 市值 + 有息负债 - 现金

行业参考倍数（成熟期）：
消费品龙头：15~25倍
工业制造：8~15倍
医疗：15~30倍
科技互联网：20~40倍
周期类（钢铁/化工）：5~10倍
电信/公用事业：6~10倍
```

## 股息折现模型（DDM）

### 适用场景
```
适合：高派息、稳定增长的成熟公司
  银行、公用事业、高股息消费品

DDM（单阶段）：
内在价值 = D₁ / (Ke - g)
D₁ = 下一年预期股息
Ke = 股权成本
g = 股息永续增长率

例：年股息2元，Ke=10%，g=3%
→ 内在价值 = 2/(0.10-0.03) = 28.6元
```

## 估值交叉验证与综合判断

### 多方法综合估值
```
给各方法赋权，综合得出估值区间：

成长型公司：
  DCF（40%) + PEG（30%）+ EV/EBITDA（30%）

成熟稳定型：
  PE历史分位（40%）+ DDM（30%）+ PB（30%）

金融类：
  PB/ROE模型（50%）+ DDM（50%）

周期类：
  EV/EBITDA（40%）+ PB底部（40%）+ PE高点入场（20%）
```

### 估值安全边际量化
```
情景分析：
悲观情景（概率20%）：内在价值 = X元
基准情景（概率60%）：内在价值 = Y元
乐观情景（概率20%）：内在价值 = Z元

期望内在价值 = 0.2X + 0.6Y + 0.2Z

买入价格 ≤ 期望内在价值 × 0.75（留25%安全边际）
卖出价格 ≥ 期望内在价值 × 1.10（接近合理价格减持）
```
