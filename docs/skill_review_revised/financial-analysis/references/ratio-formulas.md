# 核心财务比率公式速查表

## 一、盈利能力

```
毛利率 = (收入 - 营业成本) / 收入
净利率 = 净利润 / 收入
EBITDA利润率 = EBITDA / 收入
EBITDA = 净利润 + 所得税 + 利息费用 + 折旧 + 摊销

ROE = 净利润 / 平均股东权益
ROA = 净利润 / 平均总资产
ROIC = EBIT × (1 - 税率) / (权益 + 有息负债)

扣非净利润率 = (净利润 - 非经常性损益) / 收入
现金含金量 = 经营活动现金净流量 / 净利润
```

## 二、成长性

```
收入CAGR(N年) = (期末收入/期初收入)^(1/N) - 1
净利润CAGR(N年) = (期末净利润/期初净利润)^(1/N) - 1
可持续增长率 = ROE × (1 - 股利支付率)
```

## 三、营运效率

```
应收账款周转天数 = 应收账款 / (收入 / 365)
存货周转天数 = 存货 / (营业成本 / 365)
应付账款周转天数 = 应付账款 / (营业成本 / 365)
现金转换周期(CCC) = 应收周转天数 + 存货周转天数 - 应付周转天数
  （CCC越短越好，负值说明占用上下游资金，极佳信号）

固定资产周转率 = 收入 / 固定资产净值
总资产周转率 = 收入 / 总资产
```

## 四、偿债能力

```
流动比率 = 流动资产 / 流动负债
速动比率 = (流动资产 - 存货 - 预付款) / 流动负债
现金比率 = 货币资金 / 流动负债

资产负债率 = 总负债 / 总资产
净负债率 = (有息负债 - 货币资金) / 股东权益
有息负债覆盖率(DSCR) = EBITDA / (利息+到期本金)
利息保障倍数 = EBIT / 利息费用（> 3 为安全）
```

## 五、现金流分析

```
自由现金流(FCF) = 经营活动现金净流量 - 资本支出
  资本支出 = 购置固定资产/无形资产支付的现金

股权自由现金流(FCFE) = FCF - 利息费用×(1-税率) + 净借款
企业自由现金流(FCFF) = EBIT×(1-税率) + 折旧摊销 - Capex - 营运资金增加

经营性净营运资本 = (应收账款 + 存货 + 预付款) - (应付账款 + 预收款)
营运资本变化 = 期末营运资本 - 期初营运资本（增加为资金占用，负CF）
```

## 六、估值指标

```
PE = 市值 / 净利润（或 股价 / EPS）
PEG = PE / 净利润增速（%）
PB = 市值 / 净资产（或 股价 / 每股净资产）
PS = 市值 / 收入
EV = 市值 + 有息负债 - 货币资金（企业价值）
EV/EBITDA = 企业价值 / EBITDA
EV/FCF = 企业价值 / 自由现金流（巴菲特偏爱指标）
股息率 = 每股分红 / 股价
股利支付率 = 分红总额 / 净利润
```

## 七、杜邦分析分解

```
ROE = 净利润率 × 资产周转率 × 权益乘数
    = (净利润/收入) × (收入/总资产) × (总资产/净资产)

五因素杜邦：
ROE = 税负率 × 利息负担率 × EBIT利润率 × 资产周转率 × 权益乘数
```

## 八、DCF 估值核心公式

```
内在价值 = Σ [FCFt / (1+WACC)^t] + 终值 / (1+WACC)^n

终值 = FCFn × (1+g) / (WACC - g)
  g = 永续增长率（保守取GDP增速，2%~3%）

WACC = Ke × E/(D+E) + Kd × (1-t) × D/(D+E)
  Ke = 无风险利率 + β × 市场风险溢价（CAPM）
  Kd = 税前债务成本
  t = 所得税率
```

## 九、商誉减值敏感性

```
商誉减值触发测试：
可回收价值 = max(公允价值 - 处置成本, 使用价值)
使用价值 = 未来现金流折现值

减值 = max(0, 账面价值 - 可回收价值)

安全边际检验：
减值缓冲 = (使用价值 - 商誉账面价值) / 商誉账面价值
若 < 20%，小幅假设变动即可触发减值
```
