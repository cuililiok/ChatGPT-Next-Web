# 07 财务造假红旗清单（⚠️ 优先阅读）

## 造假识别总原则

> "如果某件事看起来好得令人难以置信，那它很可能就是假的。" — 华尔街格言

**四大造假动机（GONE理论）：**
- **G**reed（贪婪）：管理层利益驱动
- **O**pportunity（机会）：内控薄弱提供空间
- **N**eed（压力）：业绩指标、债务偿还、股价维护
- **E**xposure（暴露风险低）：监管宽松、审计不严

---

## 一级红旗：立即放弃该标的

### 🚩 审计相关
```
□ 审计意见为否定意见或无法表示意见
□ 审计师因无法获取足够证据而出具保留意见
□ 在重大核查期间更换审计机构（特别是从大所换小所）
□ 审计师在审计报告签发后辞职并发声明质疑管理层
□ 持续经营疑虑段落出现（公司可能破产）
```

### 🚩 现金流与利润严重背离
```
□ 连续3年净利润持续增长，经营现金流持续下降或为负
□ 净现比连续3年 < 0.3
□ 收现比连续3年 < 0.6（营收大量无法收到现金）
□ "收到的其他与经营活动有关的现金"占经营现金流 > 30%
   （虚构资金流水的常用科目）
```

### 🚩 资产负债表异常
```
□ 其他应收款长期占总资产 > 10%（大概率被大股东占款）
□ 货币资金充裕（> 总资产20%）但仍大量借入短期借款
   且利息支出显著高于正常水平
□ 存货单价持续高于市场价格（存货虚增）
□ 在建工程多年不转固，且金额巨大
□ 预付款项异常巨大，且对象为非知名供应商
```

### 🚩 关联方风险
```
□ 大量资金通过关联方循环转移（明股实债结构）
□ 关联交易定价显著偏离市场公允价值
□ 大股东通过上市公司为其关联企业提供巨额担保
□ 频繁的关联方资产注入，且定价明显偏高
```

## 二级红旗：深度核查，需要充分解释

### ⚠️ 收入质量问题
```
□ 营收增速持续远超行业均值（2倍以上），但竞争对手未感受到压力
□ 期末（Q4/12月）收入占全年比例异常高（> 40%）
□ 应收账款增速连续2年超过营收增速 > 20pct
□ 前5大客户集中度突然大幅提升（单一客户占比 > 50%）
□ 主要客户为成立时间短的小公司或关联方疑似公司
□ 营收增长但现金周转天数大幅延长（回款恶化）
```

### ⚠️ 盈利质量问题
```
□ 毛利率显著高于行业均值（超 > 15pct），但找不到合理解释
□ 研发费用资本化比例 > 60%
□ 折旧政策较同行过于宽松（使用年限明显偏长）
□ 非经常性损益占净利润比例连续 > 30%
□ 政府补贴占净利润 > 20%（补贴依赖）
□ 毛利率与净利率变化方向相反（费用处理异常）
```

### ⚠️ 管理层行为信号
```
□ 控股股东/实控人持续大额减持（特别是在业绩"靓丽"时）
□ 核心管理层集中离职（CFO/审计委员会成员）
□ 董事长/CEO年薪与公司规模严重不匹配（过高或过低均值得关注）
□ 公司频繁进行与主业无关的多元化扩张
□ 管理层回避分析师具体追问，答复语焉不详
□ 公司公告前后股价异动（内幕交易迹象）
```

### ⚠️ 资本运作异常
```
□ 频繁定向增发+承诺高业绩+后期业绩变脸
□ 并购溢价率 > 300%（3倍以上PB）且无有力协同说明
□ 高溢价收购后，创始团队通过对赌完成业绩后套现离场
□ 频繁通过借款→增资→收购→商誉的方式虚增资产
□ 上市募集资金频繁变更用途
```

## 中国A股特有造假手法

### 典型手法一：通过关联方虚构收入
```
特征：
- 关联方采购公司产品，但无真实业务需求
- 资金通过多次划转后回到上市公司（资金体外循环）

识别：
- 查工商登记，识别隐性关联方
- 对比主要客户的经营规模与采购额是否匹配
- 关注"预收款"的来源（真实预售 vs 虚假资金循环）
```

### 典型手法二：存货/固定资产虚增
```
特征：
- 存货/在建工程金额虚增，对应资产负债表虚增资产
- 同时在应付账款或其他负债虚构支出来平衡

识别：
- 存货数量×单价 vs 行业价格指数验证
- 实地调研产能规模与报表数据匹配性
- 询价第三方供应商验证采购价格
```

### 典型手法三：体外资金池
```
特征：
- 通过供应商/经销商建立体外资金池
- 在需要时注入上市公司，制造现金流
- 常见于"业绩承诺期"内的并购标的

识别：
- 供应商/客户的实质经营规模
- 与关联方的资金往来是否频繁、金额是否整数
- 银行流水与销售合同的匹配性
```

### 典型手法四：大股东占款
```
特征：
- 上市公司大量资金被大股东以"拆借"方式占用
- 体现在其他应收款科目中

识别：
- 其他应收款 / 总资产 > 5% 即预警
- 附注中是否披露对控股股东的债权
- 利息收入是否与大额其他应收款匹配
```

## 造假概率评分模型（Beneish M-Score 简化版）

```
8项指标评分（每项异常得1分）：
① 应收账款指数（当年DSRI/上年DSRI > 1.465）
② 毛利率指数（当年 < 上年毛利率，且GMI < 1）
③ 资产质量指数（非流动资产/总资产比变化）
④ 营收增长指数（当年/上年营收 > 行业均值 > 2）
⑤ 折旧率指数（折旧率降低）
⑥ 销售管理费用指数（占比上升）
⑦ 杠杆指数（有息负债占比上升）
⑧ 净现比（< 1）

得分 ≥ 4：造假概率显著偏高（Beneish原始M-score > -1.78）
得分 2~3：需要重点关注
得分 ≤ 1：相对清洁
```

## 造假企业的最终命运参考

```
A股典型案例教训：
康美药业：货币资金虚增近300亿元
  → 信号：货币资金巨额+大量借款+财务费用高
  
瑞幸咖啡：虚增营收约22亿元
  → 信号：门店经济模型不成立，FCF持续为负
  
乐视网：多元化烧钱+关联方占款
  → 信号：现金流-利润长期背离，控股股东持续质押减持
  
辅仁药业：货币资金受限+分红无法兑付
  → 信号：货币资金充裕却无法正常分红
```

> 📌 **结论**：遇到二级红旗时，要求管理层给出完整解释；遇到一级红旗时，无论估值多便宜，直接放弃。财务造假的代价是本金归零。
