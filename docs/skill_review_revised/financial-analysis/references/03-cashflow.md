# 03 现金流透视分析

## 现金流量表解读框架

### 经营活动现金流深度分析

```
经营现金流与净利润差异分析：

差异来源拆解：
① 非现金费用加回（折旧/摊销/减值）→ 正常，规模=D&A
② 营运资本占用变化 → 核心关注点
③ 其他非现金损益（公允价值变动等）→ 注意方向

营运资本变化解读：
应收账款增加 → 现金流出（客户欠款增加）
存货增加     → 现金流出（备货增加）
应付账款增加 → 现金流入（延迟付款给供应商）

最理想的营运资本结构：
→ 应付账款远大于应收账款（占用供应商，而非被占用）
→ 代表企业：零售超市、白酒、茅台模式
```

### FCF（自由现金流）深度计算

```
FCF计算方式对比：

方式一：简单法
FCF = 经营活动现金流 - 资本支出（购置固定资产/无形资产）

方式二：精确法（FCFF）
FCFF = NOPAT + D&A - 营运资本增加 - 资本支出

方式三：FCFE（股权自由现金流）
FCFE = FCFF - 利息费用×(1-税率) + 净借款

投资者实用判断：
FCF / 净利润 > 1.2  → 卓越（低资本消耗）
FCF / 净利润 = 0.8~1.2 → 良好
FCF / 净利润 = 0.5~0.8 → 一般（重资产或成长期）
FCF / 净利润 < 0.5  → 警惕（资本黑洞型业务）
FCF 持续为负       → 烧钱模式，验证商业模式
```

### FCF Yield（自由现金流收益率）
```
FCF Yield = FCF / 市值

> 5%  → 估值合理或低估（类似买入一个年化5%的实业）
> 8%  → 显著低估（稀有）
< 2%  → 估值偏贵或成长溢价高

与债券收益率比较（费雪方程思路）：
FCF Yield > 10年期国债收益率×2 → 股票相对债券有吸引力
```

## 资本支出分析（CapEx）

### CapEx性质分类
```
维持性CapEx（维持现有竞争力）：
→ 更换老化设备、日常维护
→ 可近似用折旧金额估计

成长性CapEx（扩大产能/市场）：
→ 新建产线、扩张门店、新市场投入
→ 代表未来增长投资

判断方法：
如果营收停滞但CapEx仍高 → 多为维持性CapEx，效率低下
如果营收高速增长且CapEx高 → 正常，验证转化效率
```

### CapEx强度与商业模式质量
```
CapEx / 折旧 < 1.1 → 轻资产模式（最受投资者青睐）
  代表：软件公司、品牌消费品
  
CapEx / 折旧 = 1.1~2 → 适度资本投入（成长性扩张）
  代表：消费连锁、医疗设备

CapEx / 折旧 > 2 → 重资产扩张（需验证ROIC）
  代表：制造业、电力、地产

CapEx / 折旧 > 3 → 极重资产（需要非常高的行业壁垒才值得投资）
  代表：航空、造船、重化工
```

## 现金流预警信号系统

### 一级预警（立即深查）
```
🚨 净现比连续3年 < 0.5
🚨 经营现金流持续为负，而净利润持续为正
🚨 收现比连续下降至 < 0.7
🚨 "收到的其他与经营活动有关的现金"异常大额
   （常用于虚构现金流的科目）
🚨 投资活动产生的现金流中，大量来自"收回投资"
   而业务并无实质性收缩
```

### 二级预警（需要解释）
```
⚠️ 净现比连续2年 < 0.8
⚠️ FCF突然从正转负（排除正常大规模扩张）
⚠️ 应收账款周转天数连续增加
⚠️ 存货周转天数大幅上升
⚠️ 经营现金流虽然为正，但来自预收款大幅增加
   而非真实销售回款
```

## 现金转化周期（CCC）分析

```
CCC = 应收账款周转天数（DSO）
    + 存货周转天数（DIO）
    - 应付账款周转天数（DPO）

解读：
CCC < 0 → 极优秀（零售/白酒类，先收钱后付款）
CCC = 0~30天 → 优秀
CCC = 30~60天 → 正常
CCC > 90天 → 营运资金占用大，需要更多融资

CCC改善 → 运营效率提升，正向信号
CCC恶化 → 运营效率下降或回款困难，负向信号

明星案例：
茅台 CCC ≈ 负值（经销商预付款模式）
制造业 CCC 通常60~120天
```

## 分红与股东回报分析

```
派息率（分红/净利润）分析：
> 70%  → 成熟稳定型，现金牛企业（适合DDM估值）
40%~70% → 均衡型（兼顾成长与回报）
< 30%  → 成长型（再投资优先）或现金流紧张

自由现金流分红覆盖率：
= FCF / 实际分红金额
> 1.5  → 分红可持续且有提升空间
< 1.0  → 分红不可持续（用借债分红→危险信号）

股票回购分析：
回购 + 注销股份 → 正向信号（管理层认为股价低估）
回购 + 不注销  → 可能用于股权激励，价值中性
```
