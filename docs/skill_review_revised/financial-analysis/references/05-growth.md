# 05 成长性与可持续性分析

## 成长质量评估框架

### 三类成长来源及质量排序

```
第一类：市场扩容（最高质量）
→ 行业整体增长，公司随行业增长
→ 可持续性强，无需抢夺竞争对手份额
→ 分析：行业增速（需结合industry-analysis skill）

第二类：市占率提升（高质量）
→ 公司增速 > 行业增速
→ 在吃竞争对手的份额
→ 需要强护城河支撑（成本、品牌、技术、渠道）
→ 分析：市占率变化趋势

第三类：并购整合（中低质量）
→ 通过收购实现营收增长
→ 需验证：整合协同效应是否实现
→ 风险：商誉减值、文化冲突、管理摊薄
→ 分析：有机增速 = 总增速 - 并购贡献

伪成长（危险）：
→ 靠关联交易虚增营收
→ 靠提前确认收入冲高当期业绩
→ 靠政府补贴维持利润增长
```

### 成长持续性判断模型

```
可持续增长率（内部融资能力）：
SGR = ROE × (1 - 派息率)

含义：在不外部融资的情况下，公司能支撑的最大增长速度

例：ROE=20%，派息率=30%
→ SGR = 20% × 70% = 14%
若实际增速 > 14%，需要外部融资（股权/债权）
若长期依赖外部融资 → 商业模式自我造血能力不足

再投资效率：
再投资回报率 = 净利润增量 / 净资产增量
连续5年 > 15% → 优秀（再投资产生高回报）
< 8%          → 差（成长在销毁价值）
```

## 驱动成长的量价拆解

### 量的维度（市场渗透）
```
现有市场份额分析：
当前市占率 → 上限（理论天花板）
近3年市占率变化趋势（加速/减速/见顶）

新市场拓展：
地理扩张（省份/国家覆盖度）
渠道扩张（线上/线下/下沉市场）
客群扩张（品类延伸/年龄层延伸）

产能扩张可信度：
在建工程转固节奏 vs 营收增长节奏
产能利用率水平（< 70% = 产能过剩；> 90% = 扩产迫切）
```

### 价的维度（定价能力）
```
提价能力验证：
历史提价记录（频率、幅度、市场接受度）
价格弹性：提价后销量是否下滑？
单价趋势 = 营收增速 - 销量增速

提价空间判断：
产品差异化程度（标准化 vs 品牌化）
消费者对产品总消费支出的占比（占比越低，提价越容易）
竞品价格对比（相对溢价空间）

结构升级（隐形提价）：
高端产品线占比提升
产品组合向高毛利方向迁移
```

## 研发投入与创新能力

### 研发质量评估
```
研发费用率 vs 行业标准（见02-profitability.md）

研发产出验证：
专利数量 & 质量（发明专利 vs 实用新型）
新产品贡献收入占比
新产品毛利率 vs 老产品（高则说明成功）
管线/在研项目进展（医药/半导体行业重点）

研发人员分析：
研发人员占比 & 变化趋势
人均研发投入（与薪酬水平结合）
核心研发人员离职率

资本化研发风险：
资本化/总研发支出 > 50% → 深查合理性
研发资本化金额异常增长 → 利润美化嫌疑
```

## 管理层与公司治理评估

### 管理层能力评估（定性）
```
历史兑现度：
过去5年管理层给出的业绩指引 vs 实际结果
战略规划的连贯性（朝令夕改 vs 专注主业）
资本分配记录（并购是否创造价值？回购时机是否合理？）

激励机制：
管理层持股比例（创始人团队持股 > 5% 通常是正面信号）
股权激励计划设计（行权条件是否有挑战性？）
薪酬结构（固定/浮动比例，是否与股东利益绑定）
```

### 公司治理风险清单
```
结构性风险：
□ 双重股权结构（同股不同权，创始人控制权过大）
□ 独立董事占比 < 1/3
□ 审计委员会无独立董事主导
□ 外部审计师非独立（同一会计师事务所长期 > 10年）

行为风险信号：
□ 管理层频繁买卖公司股票（特别是大额减持）
□ 高管薪酬与公司业绩严重不匹配
□ 大量关联交易，定价不透明
□ 公司频繁修改会计估计和政策
□ 投资者关系不透明，回避分析师提问
```

## 第二增长曲线评估

### 新业务可行性分析框架
```
对照第一增长曲线的核心能力：
① 与主业的协同性（技术/渠道/客户共享度）
② 新业务的商业模式可行性（参考industry-analysis skill）
③ 管理层在新领域的经验积累

新业务投入产出分析：
新业务收入/总收入趋势（规模化速度）
新业务毛利率趋势（是否在改善）
新业务对整体ROE的稀释程度（短期可接受，长期需转正）

判断标准：
3年内：新业务可以亏损，但需验证商业模式
5年内：新业务应贡献正毛利，路径清晰
10年内：新业务应实现整体盈利，不再是主业的负担
```
