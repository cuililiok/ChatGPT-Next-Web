# 财务造假深度识别指南

## 一、造假动机识别（先问Why）

造假通常有以下动机，存在动机时需提高警惕：
- 股权质押比例高（大股东需维持股价，防爆仓）
- 业绩承诺期内（并购后3年，造假完成承诺）
- IPO/定增/发债前后（融资需要包装利润）
- 管理层持股行权条件挂钩业绩
- 债务即将到期，需再融资

## 二、收入造假识别

### 虚构收入特征
```
信号：
  收入增长 >> 行业增速，无合理市场份额解释
  收入增长 >> 同期上下游企业增速
  收入增长，但员工数/产能未相应扩张
  季末/年末收入占比异常高（"冲量"特征）

验证方法：
  ① 税务勾稽：增值税销项税额 ÷ 收入 = 适用税率（应跨年稳定）
     若收入上升但含税率下降 → 部分收入可能不含税/虚构
  ② 水电费用勾稽：制造业收入大幅增长但水电支出平稳 → 产能未提升
  ③ 运输费用：收入翻倍但快递/运输成本未增 → 货物没有实际流动
```

### 提前确认收入
```
信号：
  大量"发出商品"科目（货已发但未确认收入暂存此处）
  年底应收账款突增，次年初大额冲回
  长期合同中收入确认节点不透明
```

## 三、资产造假识别

### 应收账款造假
```
高风险特征：
  账龄结构恶化：1年以上占比逐年提升
  坏账计提比例低于同行（政策过于宽松）
  大额应收集中在少数关联或神秘客户

深度核查：
  坏账准备充分性：1年以上应收，80%+计提才算保守
  应收账款周转天数 vs 同行历史走势
```

### 其他应收款（最危险科目）🚩
```
高风险特征：
  金额巨大（超过净资产10%）
  对象为关联方或来路不明的公司
  账龄较长，无明确回收时间表
  与主营业务毫无关联

本质：通常是大股东或关联方占用上市公司资金的通道
```

### 商誉减值风险
```
风险信号：
  商誉 / 净资产 > 30% → 减值风险区间
  被收购公司业绩承诺完成率——承诺期结束后是否"变脸"
  减值测试假设收入增速明显超过行业均值 → 可能过于乐观

规律：承诺期3年内100%完成，第4年大额减值 = 典型的"业绩承诺游戏"
```

## 四、现金流造假识别

### 经营现金流伪造
```
常见手法：
  将融资性应收款归入经营现金流
  关联方垫付货款，制造虚假回款
  应付账款大幅增加而采购未增（占用供应商资金美化现金流）

识别：
  经营现金流增速 >> 收入增速（正常情况应接近）
  存在大量"其他"现金流项目且无详细说明
```

## 五、定量工具

**Beneish M-Score 8因子模型：**
```
若M-Score > -1.78，企业可能存在盈余操纵
核心因子：
  DSRI  应收账款指数（>1 = 应收账款增速超过收入）
  GMI   毛利率指数（<1 = 毛利率下降）
  AQI   资产质量指数（>1 = 非流动资产比例上升）
  DEPI  折旧指数（>1 = 折旧率下降，可能少计折旧）
  TATA  应计项目占比（净利润中非现金部分占比过高）
```

**Altman Z-Score 破产预警：**
```
Z = 1.2X1 + 1.4X2 + 3.3X3 + 0.6X4 + 1.0X5
  X1 = 营运资本/总资产
  X2 = 留存收益/总资产
  X3 = EBIT/总资产
  X4 = 股权市值/总负债账面价值
  X5 = 收入/总资产

Z > 2.99 → 安全区
1.81~2.99 → 灰色区
Z < 1.81 → 破产高风险区
```
