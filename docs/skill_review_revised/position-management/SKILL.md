# 仓位管理与组合构建框架

## 核心前提：仓位是确定性的函数

> 仓位的大小，不应该由你对股价涨跌的预期决定，而应该由你对这家公司的理解深度决定。

这是本框架所有内容的基础。所谓"仓位管理"，本质上是在问：**我对这笔投资的确定性有多高？**

---

## 第一部分：单一持仓的仓位上限

### 确定性分级

在买入前，先对这笔投资的确定性做一个诚实的评估：

| 确定性等级 | 条件 | 建议仓位上限 |
|-----------|------|------------|
| **S级：极高确定性** | 能力圈核心领域 + 商业模式清晰 + 企业文化可靠 + 估值合理 + 财务健康 + 已深入研究3年以上 | 30%～50% |
| **A级：高确定性** | 能力圈内 + 商业模式好 + 基本面扎实 + 有1～2个待观察变量 | 15%～30% |
| **B级：中等确定性** | 能力圈边缘 + 商业模式较好 + 仍有较多不确定性 | 5%～15% |
| **C级：低确定性** | 能力圈外 + 逻辑尚未完全验证 + 观察仓 | ≤5% |

**硬性规则：**
- 单一持仓超过20%，必须能回答"如果明天跌30%，我的论文有没有变化"
- 单一持仓超过30%，必须是你研究最深、最有把握的标的
- 任何时候都不应该因为"感觉会涨"而加仓，只应该因为"确定性提高"而加仓

## 第二部分：组合构建原则

### 集中还是分散？

**价值投资者的答案是：有条件的集中。**

巴菲特说过把钱最多分散到6个公司。段永平10年里只重手投了5家。他们能集中的前提是：**对这几家公司的理解远超市场平均水平。**

判断你是否应该集中的标准只有一个：
> 你的持仓数量，应该等于你真正看懂的公司数量，而不是你想分散风险的数量。

| 情况 | 建议 |
|------|------|
| 真正看懂的公司 ≤ 3家 | 集中持有这几家，其余放现金或指数 |
| 真正看懂的公司 4～8家 | 按确定性分级配置，头部2～3家占60%以上 |
| 号称看懂超过10家 | 极可能是自我欺骗，重新审视能力圈 |

### 组合内的相关性检查

分散持有5只股票，如果它们都受同一个宏观变量驱动（例如都是顺周期，或都依赖同一政策），本质上你只是持有了同一个风险的5份拷贝。

**相关性检查清单：**
- 这几家公司的盈利驱动因素是否不同？
- 如果利率大幅上升，哪些会受影响？
- 如果消费大幅萎缩，哪些会受影响？
- 如果监管收紧，哪些会受影响？
- 在同一个极端情景下，会不会同时大跌？

> 目标不是零相关，而是**不要在最坏的时候同时大幅下跌**。

## 第三部分：加仓逻辑

这是最考验投资者的地方。股价下跌时，加仓还是止损？

### 判断框架：论文有没有变化？

**买入时写下买入论文（3句话以内）：**
1. 这家公司好在哪里（商业模式/护城河）
2. 为什么现在是合适的价格
3. 什么情况下我会卖出

**股价下跌时，只问一个问题：**
> 买入论文有没有被证伪？

```
论文未变 + 价格更低 → 应该加仓（价格给了更好的机会）
论文已变 + 无论价格 → 应该卖出（发现错了马上改）
论文不确定 + 价格更低 → 不动，继续观察
```

### 加仓的节奏：分批而非一次性

即使论文未变、判断正确，股价可能还会继续跌。因此：

**建议采用"子弹分配法"：**
- 初始建仓：用计划仓位的40%～50%
- 预留弹药：30%用于第一次大幅下跌后加仓
- 保留底仓：20%～30%备用，用于出现极端低估时

**加仓触发条件（满足其一）：**
- 股价较买入成本下跌15%～20%以上，且论文未变
- 出现市场非理性恐慌（指数大跌10%以上），且个股基本面未恶化
- 公司发布了利好基本面的新信息（新产品、新客户、回购等）

**禁止加仓的情况：**
- 买入论文已经开始动摇但还没完全证伪（这是最危险的情绪陷阱）
- 加仓后该仓位将超过对应确定性等级的上限
- 是因为"跌了这么多应该反弹了"而加仓（这是在预测股价，不是在投资）

## 第四部分：减仓与卖出逻辑

### 三种应该卖出的情况

**1. 论文证伪（最重要，必须执行）**
- 商业模式出现不可修复的破坏
- 企业文化/管理层暴雷（欺诈、重大关联交易、持续毁价值并购）
- 护城河正在系统性侵蚀，不是暂时的

> 发现错了马上改，不管多大代价都是最小代价。——段永平

**2. 找到了更好的机会**
- 机会成本是卖出的合理理由
- 但标准要一致：新标的的确定性，必须显著高于当前持仓
- 不能因为想"换个新鲜的"而卖出

**3. 估值严重透支**
- 股价已远超合理估值，内在价值已被透支多年
- 但伟大公司要惜卖——找到一家真正优秀的公司极难，不要为一点差价轻易卖出

### 不应该卖出的情况

- 股价短期大跌（论文未变的情况下）
- 市场悲观情绪蔓延（这是持有的理由，不是卖出的理由）
- 短期业绩低于预期（要区分是暂时的还是结构性的）
- 有人告诉你"涨太多了该卖了"（问他们的论文是什么）

## 第五部分：现金仓位的管理

### 现金不是在等待，现金是一个选项

持有现金的合理理由：
- 目前能力圈内没有足够确定性的标的处于合适价格
- 需要为未来更大的机会保留弹药
- 不是因为"觉得市场要跌"而持现金（择时是很难成功的游戏）

### 满仓主义的前提

段永平是"满仓主义者"，但他的满仓有前提：
> 满仓主义不意味着就一定要满仓，没有合适投资标的时（包括价格过高时）是可以拿着现金的。

**现金比例参考：**
| 情况 | 现金建议 |
|------|---------|
| 能力圈内有1～2个高确定性标的处于合适价格 | 0%～10%（充分投入） |
| 有标的但价格偏高，等待更好机会 | 20%～40% |
| 能力圈内暂时找不到合适标的 | 40%以上，耐心等待 |
| 市场整体高估，好公司也贵 | 视情况，可持有较多现金 |

## 第六部分：常见错误与纠正

| 错误 | 根本原因 | 纠正方式 |
|------|---------|---------|
| 持仓太分散（20只以上）| 用数量掩盖不确定性 | 问自己：每只股票你能讲清楚买入论文吗？ |
| 赚钱的卖，亏钱的留 | 处置效应 | 每只股票独立判断，与成本无关 |
| 跌了不敢加，涨了反而追 | 把股价当信号 | 回归论文：基本面有没有变化？ |
| 仓位平均化（每只10%）| 对所有判断同等对待 | 好的判断应该重仓，差的判断应该少仓或不仓 |
| 持仓太集中但研究不足 | 把胆大误认为有把握 | 确定性不够的情况下集中是赌博 |
| 频繁换仓 | 对短期波动过度反应 | 问自己：3年后这次操作重要吗？ |

## 组合健康度自查清单

每季度检查一次：

```
【持仓结构】
□ 每只持仓都有3句话以内的买入论文？
□ 最大单一持仓是否超过对应确定性等级的上限？
□ 持仓总数是否在你真正看懂的公司数量以内？

【相关性检查】
□ 是否有两只以上持仓在同一极端情景下会同时大跌？
□ 组合的整体风险是否集中在某一个行业或宏观变量上？

【现金弹药】
□ 是否为潜在机会保留了足够的弹药？
□ 持有现金的理由是"没有好机会"而非"预测市场要跌"？

【论文状态】
□ 每只持仓的买入论文，目前是否仍然成立？
□ 是否有持仓的论文已经动摇但还没有处理？（最危险的状态）

【综合判断】
组合整体状态：健康 / 需要调整 / 需要大幅重构
```

---

## 一句话总结

> 仓位大小 = 你对这家公司的理解深度。
> 组合构建 = 把有限的子弹，分配给你最有把握的几个判断。
> 加仓减仓 = 只看论文有没有变化，与股价涨跌无关。
