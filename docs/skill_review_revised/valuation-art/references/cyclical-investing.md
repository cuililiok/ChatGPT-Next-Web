# 周期股投资精华框架

综合霍华德·马克斯《周期》、彼得·林奇《彼得·林奇的成功投资》、吉姆·罗杰斯《热门商品投资》及格雷厄姆《证券分析》中关于周期性行业的核心方法论。

---

## 一、周期定位：你现在在哪里？（马克斯框架）

投资周期股，最重要的不是给公司估值，而是先判断**宏观周期位置**。马克斯将周期分为三层，层层嵌套：

### 三层周期

```
经济周期（GDP / 工业产出）
    └─ 信贷周期（银行放贷意愿 / 利差水平）
          └─ 市场情绪周期（风险偏好 / 估值水平）
```

**核心洞见**：信贷周期是放大器。经济上行时信贷宽松放大繁荣，下行时信贷收缩放大衰退。周期股的暴涨暴跌，往往是信贷周期在产业层面的镜像。

### 周期位置判断清单

**可能处于周期顶部的信号：**
- 行业产能利用率 >90%，新项目大量上马
- 企业普遍乐观，分析师一致上调盈利预测
- 并购活跃，溢价越来越高
- 行业P/E看起来"很低"（利润在高点）
- 媒体开始报道"超级周期"、"这次不一样"

**可能处于周期底部的信号：**
- 大量企业亏损，产能开始出清（停产、破产、兼并）
- 分析师纷纷下调评级，机构持仓创历史低位
- 行业P/E极高或无意义（利润在低点）
- 新建产能投资几乎为零
- 媒体在报道"行业寒冬"、"看不到尽头"

> 马克斯的核心方法论：你不需要精确知道周期顶部或底部在哪里，只需要判断**大概在左半段还是右半段**，然后相应调整仓位。

## 二、林奇的周期股操作信号

彼得·林奇将周期股单独列为一类，强调与成长股**完全相反的操作逻辑**：

### 买入信号（底部区域）

1. **库存开始去化**：渠道库存从高位回落，企业不再被迫降价促销
2. **产能退出加速**：弱势竞争者停产、破产或被兼并（供给侧收缩）
3. **分析师刚开始下调**：机构悲观情绪蔓延但底部往往在此附近（逆向信号）
4. **大宗商品价格跌破全行业边际成本**：亏损性生产不可持续，供给自然出清
5. **龙头公司现金流仍为正**：强者在底部活下去，弱者离场，竞争格局改善

### 卖出信号（顶部区域）

1. **新产能大量落地**：2~3年前的扩张投资开始释放供给
2. **客户开始建立安全库存**：需求端的一次性拉动（之后会回落）
3. **公司开始提高股息或大规模回购**（用景气期利润做顺周期操作，而非投资护城河）
4. **P/E看起来极低**（这时候正是顶部！利润虚高）
5. **竞争对手开始宣布扩产计划**

**林奇原则**：周期股上，好消息出尽是卖出时机，坏消息出尽是买入时机。与成长股逻辑完全颠倒。

## 三、供需框架：大宗商品的定价逻辑（罗杰斯框架）

适用于能源、金属、农产品等大宗商品驱动型周期股。

### 供给分析（比需求更重要）

大宗商品的长周期由**供给端**主导，因为需求相对稳定，而供给的投资周期长、弹性小。

关键问题：
- 主要产区的产能利用率是多少？
- 新建产能的资本开支周期有多长？（铜矿：8~10年；LNG：5~7年）
- 是否有政策/地缘因素限制供给扩张？
- 现有产能的老化速度如何（自然衰减率）？

### 需求分析

- 中国工业化/城镇化阶段对大宗商品的结构性拉动
- 能源转型对不同品种的差异化影响（铜、锂利好；煤炭、燃油受压）
- 库存周期的短期扰动（通常3~18个月）vs 资本开支周期的长期趋势（3~10年）

### 罗杰斯的核心判断框架

```
长期牛市条件 = 供给长期不足 + 需求结构性增长 + 资本开支持续低迷
```

当大宗商品价格长期低迷时，勘探/开采投资被压缩；数年后供给缺口出现，价格大幅上涨。这是他最核心的判断逻辑——**看资本开支周期，不看当前价格**。

## 四、周期股专用估值方法

### 1. 穿越周期标准化盈利（Graham / 马克斯）

源自格雷厄姆《证券分析》，是周期股估值的基石方法。

**计算方式**：
```
标准化EPS = 过去完整周期（7~10年）的平均EPS
穿越周期P/E = 当前股价 / 标准化EPS
```

**判断标准**：
- 穿越周期P/E < 历史均值的70% → 可能低估区域
- 穿越周期P/E > 历史均值的130% → 可能高估区域

**注意**：需要确保所取周期包含完整的一个高峰和一个低谷，避免截断偏差。

---

### 2. P/B + ROE回归均值法

逻辑：周期股的ROE会均值回归，当前P/B反映了对未来ROE的隐含预期。

```
隐含ROE = P/B × COE（股权成本）

若 隐含ROE << 历史均值ROE → 市场过度悲观 → 可能低估
若 隐含ROE >> 历史均值ROE → 市场过度乐观 → 可能高估
```

---

### 3. 替代成本法（Replacement Cost）

逻辑：当市值 < 新建同等产能的成本时，买股票比建工厂便宜，市场应向替代成本收敛。

适用场景：钢铁、水泥、普通大宗化工（如化肥、煤化工大宗品）、矿山、航运等重资产 + 同质化产品行业

> ⚠️ **不适用：精细化工、特种化工、化工新材料、高端涂料、电子化学品、医药 CDMO**
> 这类公司的真实价值主要来自**客户认证壁垒（FDA/汽车 Tier 1/电子大厂的严苛审核流程）**和
> **工艺 know-how（专利保护期外的工艺秘密）**。新建同等产能的"替代成本"远低于这些
> 无形资产的真实价值——某家精细化工龙头工厂的"替代成本"可能只是其市值的 5–10%，
> 但这并不意味着市值高估，因为竞争对手即使建好工厂也通不过下游客户的认证。
>
> 这类公司应改用消费品/科技股框架（见 SKILL.md Step 2 护城河分析），按 ROIC 持续高于 WACC
> 的程度给定价（PE 通常 25–40x，远高于普通化工 5–12x）。

步骤：
1. 估算新建同等规模产能的总投资（参考行业近期并购/新建项目披露）
2. 计算"替代成本倍数" = 当前EV / 替代成本
3. 历史上替代成本倍数 <0.6 往往是强烈底部信号

---

### 4. 大宗商品价格敏感性分析

对于强周期股，必须明确盈利对商品价格的弹性：

| 商品价格变动 | 对EBIT的影响 | 对EPS的影响（考虑杠杆放大） |
|------------|------------|--------------------------|
| +10% | ? | ? |
| -10% | ? | ? |
| 跌至边际成本 | ? | 亏损 |

**操作方式**：建立三情景模型
- 悲观：商品价格跌至行业90th percentile成本（即90%产能亏损线）
- 基准：当前期货曲线隐含价格
- 乐观：历史均值价格 × 1.2

**关键判断**：悲观情景下公司是否有足够流动性撑过低谷（净债务/EBITDA < 3x，或有充足授信额度）

## 五、周期股的资产负债表要求

周期股对资产负债表的要求远高于成长股，因为低谷期可能持续2~3年。

**底线要求（缺一不可）：**
- 净债务/EBITDA（峰值利润时）< 2x（保守派要求 <1.5x）
- 短期债务/总债务 < 30%（避免低谷期再融资风险）
- 有稳定的银行授信额度（承诺额度，非用量）
- 固定成本占总成本比例越低越好（成本弹性越高越能扛住低谷）

**高杠杆周期股的死亡路径**：
```
价格下跌 → 利润骤降 → 信用评级下调 → 融资成本上升 / 银行收贷 → 
被迫低价出售资产 → 净资产缩水 → 股价加速下跌
```

这是为什么即使判断对了周期底部，选错了高杠杆公司也会血本无归。

## 六、化工股专项说明

化工行业横跨"强周期"到"弱周期"，分析前必须先做分类判断，否则套错框架。

### 第一步：判断属于哪类化工

| 类型 | 典型品种 | 周期强度 | 适用框架 |
|------|---------|---------|---------|
| **大宗基础化工** | 乙烯、甲醇、PVC、纯碱、TDI/MDI | 强周期 | 本文件全部适用 |
| **化工中游（通用）** | 苯乙烯、己内酰胺、丙烯酸 | 中等周期 | 本文件适用，但周期更短 |
| **精细/特种化工** | 电子化学品、高端涂料、农药原药、功能性添加剂 | 弱周期 | 用消费品/科技股框架（见SKILL.md Step 2护城河分析） |
| **化工新材料** | 碳纤维、PVDF、POE | 成长属性为主 | 成长股框架为主，周期框架为辅 |

**判断特种化工的标志**：
- 产品需要客户认证（换供应商成本高）
- 毛利率 >30% 且跨周期稳定
- 研发投入占收入 >3%
- 主要客户是品牌消费品或高端制造商

符合上述特征的化工公司，**不应套用周期股框架**，应按护城河逻辑估值。

---

### 第二步：价差分析（比商品价格更重要）

化工公司的利润核心是**产品价格与原料成本之间的价差**，而非产品价格绝对值。

**为什么价差比价格更重要**：
- 原油涨价时，乙烯装置的裂解价差可能反而收窄（下游议价能力弱）
- 煤价下跌时，煤化工企业利润可能大幅改善（即使产品价格也在跌）
- 只看产品价格，会系统性误判化工公司盈利弹性

**价差敏感性分析模板**（替代通用框架的"商品价格三情景"）：

```
关键价差 = 主产品均价 - 主要原料成本（折算单位）

悲观情景：价差收窄至近10年10th percentile
基准情景：近5年均值价差
乐观情景：近10年90th percentile价差
```

需要分析的关键价差因产品而异：
- **乙烯链**：乙烯-石脑油价差、PE/PP-乙烯价差
- **煤化工**：甲醇-煤炭价差、尿素-煤炭价差
- **氯碱**：PVC-电石/乙烯价差，烧碱单独定价
- **MDI/TDI**：成品-苯/甲苯原料价差

**一体化程度的影响**：
原料自给率越高，价差波动对公司利润的影响越小，估值应给予稳定性溢价。全产业链一体化企业（如万华化学）在周期底部的抗跌性显著强于纯下游加工企业。

---

### 第三步：化工行业的供给特殊性

**产能建设周期比金属短**：
- 化工装置：2~3年可投产
- 有色/钢铁矿山：5~10年
- 这意味着化工供给反应更快，周期持续时间通常更短（顶部信号出现后留给操作的窗口更窄）

**中国化工的特殊供给约束：环保与安全监管**

中国化工行业存在一个本土特有的供给收缩机制：
- 化工园区整合政策（园区外企业强制搬迁/关停）
- 安全生产事故后的行业整治（如2019年响水爆炸引发的大规模停产）
- 双碳目标对高耗能化工品的产能限制

这类政策性供给收缩有别于市场化产能出清，具有**突发性强、持续时间不确定**的特点，分析时需单独评估政策敞口。

**判断政策性供给收缩是否可持续的问题**：
- 被限制的产能是否有替代品或进口来源？
- 限产是否导致下游成本上升引发政策逆转压力？
- 龙头企业是否从限产中受益（合规成本更低、可承接份额）？

---

### 化工股分析补充清单

在标准周期股分析清单基础上额外确认：

**分类判断**
- [ ] 该化工品属于大宗/通用/精细/新材料哪类？
- [ ] 是否符合特种化工标志（认证壁垒、稳定毛利率、研发投入）？

**价差分析**
- [ ] 识别出1~2个关键价差指标
- [ ] 当前价差处于历史分位数的哪个位置？
- [ ] 原料自给率/一体化程度如何？

**供给分析**
- [ ] 新建产能的建设周期是多长？在建产能规模？
- [ ] 是否存在环保/安全监管的供给约束？该约束的可持续性？
- [ ] 行业CR5集中度趋势（分散→集中意味着定价权改善）

**底部信号（化工特有）**
- [ ] 价差是否已低于行业边际成本（50%以上产能亏损）？
- [ ] 高成本产能是否开始停产检修（非计划性）？
- [ ] 主要港口库存是否处于历史高位并开始去化？

## 七、周期股分析完整清单

分析一家周期股时，在标准五步框架基础上额外完成：

**周期定位（先做这个）**
- [ ] 行业当前处于周期哪个阶段？（马克斯顶部/底部信号清单）
- [ ] 产能利用率水平？新建产能投资趋势？
- [ ] 大宗商品价格相对于行业边际成本的位置？
- [ ] 机构持仓和分析师评级的情绪极端程度？

**公司筛选**
- [ ] 资产负债表强度（能否扛过2~3年低谷）
- [ ] 成本曲线位置（是否处于行业低成本端）
- [ ] 管理层的周期管理能力（历史上如何在高峰期控制扩张冲动）

**估值**
- [ ] 穿越周期P/E（完整周期均值EPS）
- [ ] P/B + 隐含ROE合理性
- [ ] 替代成本倍数（重资产行业）
- [ ] 三情景大宗商品价格敏感性（化工股改为价差敏感性，见第六节）
- [ ] 悲观情景下的流动性压力测试

**买卖决策**
- [ ] 林奇买入信号核查（库存去化？产能退出？）
- [ ] 林奇卖出信号核查（新产能落地？分析师转乐观？）
- [ ] 安全边际：当前价格 vs 穿越周期估值的折价幅度
