## 金融（证券 / 经纪商 / 资管）

> ⚠️ 证券公司是**金融周期股**：ROE高度依赖市场行情，历史波动幅度可达1%~20%+。单年ROE输入任何估值公式几乎必然误导。以下所有方法均要求先完成ROE正规化。

### 估值指标参考范围

| 指标 | 典型范围 | 备注 |
|------|---------|------|
| P/B | 0.7~2.0x | 高度周期依赖；熊市底部接近或跌破1x；牛市顶部可达2x+ |
| P/E | 10~25x | 周期顶部P/E偏低（利润虚高），底部极高甚至无意义，**慎用** |
| 正规化ROE | 5~9%（A股券商历史均值区间）| 单牛市年份可达15~20%，不代表常态 |

### 首选估值方法

**FCFE DCF 为主方法**（Damodaran《The Dark Side of Valuation》明确推荐证券/资管公司使用股权自由现金流折现）。P/B-ROE 和 DDM 作为交叉验证，不作主估值。

P/B-ROE 公式 `PB = (ROE - g) / (Ke - g)` 对证券公司的隐含前提：
- ROE 在永续期内保持稳定 → **证券公司ROE可从1%波动到20%+，前提不成立**
- 分红率稳定 → 实际上牛熊市分红政策随利润波动
- **结论：P/B-ROE 可辅助交叉验证，不能单独作为主估值方法**

DDM 对分红率假设极度敏感（分红率从35%提至50%，估值提升约40%），仅供参考。

### ROE 正规化方法（必须执行）

建议使用全历史数据（≥10年，含完整牛熊周期），采用**指数衰减加权均值（半衰期5年）**：

- 权重公式：`0.5^((当前年 - 年份) / 5)`
- 优点：保留全周期数据（不人为删除牛市峰值或熊市谷底）；近年权重更高，反映行业新常态
- 同时列出三情景：悲观（近10年最低ROE）、基准（指数加权均值）、乐观（最近年度实际ROE）

不推荐截尾均值：对券商而言，"两端极值"正好是2008/2009/2015等关键牛市年份，截掉会系统性低估周期高点。

注意：券商不适合 EV/EBITDA（债务是经营性的）。

## 周期性行业（特殊处理）

> ⚠️ 周期股估值的核心陷阱：在周期顶部利润最高时P/E最低（看起来"便宜"），在周期底部利润最低或亏损时P/E最高甚至无意义（看起来"贵"）。**用单年P/E分析周期股几乎必然得出错误结论。**

### 通用周期股估值原则

| 方法 | 说明 |
|------|------|
| **穿越周期P/E**（Normalized P/E） | 用5~10年平均EPS代替当年EPS计算P/E，消除周期扭曲 |
| **P/B法** | 周期底部资产打折时最有参考价值；ROE回归均值时对应多少P/B |
| **EV/EBITDA（周期中性年）** | 选取接近周期中点的年份数据作为基准 |
| **替代成本法** | 当前市值 vs. 新建同等产能需要多少钱（尤其适合重资产周期行业） |
| **股息率法** | 部分周期股（如航运、资源）在底部高分红时用股息率判断底部区域 |

**何时买入周期股的信号**：P/B接近历史底部、行业产能大规模出清、亏损企业开始退出、大宗商品价格低于全行业边际成本

## 使用说明

1. 倍数本身没有意义，**必须与增长率和盈利质量挂钩**
2. 高倍数不等于高估，低倍数不等于低估——需看"为什么"
3. 跨市场比较（中国 vs 美国 vs 欧洲）需考虑利率水平差异
4. 周期性行业在周期顶部P/E最低（利润虚高），在底部P/E最高（利润虚低）——此时用P/B或EV/Sales更合适
