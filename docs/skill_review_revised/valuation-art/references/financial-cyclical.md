# 金融周期股专项框架

金融股是周期股中的特殊门类，工业周期框架的大多数工具（EV/EBITDA、替代成本法、价差分析）在此完全失效。金融公司的债务是经营性资产而非财务杠杆，必须用专属逻辑分析。

本文件覆盖：证券公司、保险公司（寿险/财险分开处理）、商业银行。

---

## 一、金融股的共同底层逻辑

### 为什么EV/EBITDA失效

工业公司：债务是融资工具，可从企业价值剥离
金融公司：债务（存款、保险负债、融资融券）是**经营性原材料**，不可剥离

因此金融股统一使用**股权价值框架**，而非企业价值框架：
- 核心估值指标：P/B（市净率）
- 核心盈利指标：ROE（净资产收益率）
- P/B与ROE的关系：当ROE > COE（股权成本）时，P/B应 >1；ROE < COE时，P/B应 <1

### 金融周期的特殊性

金融股的周期驱动与工业周期不同步，且往往**领先**经济周期：

```
经济扩张预期 → 资本市场活跃（证券受益）
             → 信贷需求上升（银行受益）
             → 保费增长（保险受益）

利率上行 → 银行NIM扩张（银行受益）
        → 保险投资收益改善（保险受益）
        → 债券承销成本上升（证券投行受压）

监管收紧 → 对具体业务线的影响各不相同（需单独分析）
```

## 二、证券公司（券商）

### 业务结构与周期敏感性

| 业务线 | 收入驱动因素 | 周期敏感性 |
|--------|------------|----------|
| 经纪业务 | 市场日均成交量（ADT） | 极高，弹性最大 |
| 投行业务 | IPO/再融资/债券承销规模 | 高，受监管政策影响大 |
| 资产管理 | AUM规模及结构 | 中等，费率有刚性 |
| 自营投资 | 股票+债券自营盘表现 | 高，且影响资产负债表质量 |
| 两融业务 | 市场杠杆水平、利率 | 高，与市场情绪高度相关 |

**收入结构判断**：经纪+投行占比越高，周期弹性越大；资管+机构业务占比高则更稳定。

---

### 证券股的周期定位信号

**底部信号（买入区域）**：
- 市场日均成交量跌至近三年均值的50%以下
- IPO暂停或大幅缩量，投行业务收入骤降
- 券商股P/B跌破1x（尤其是龙头券商）
- 两融余额持续下降，去杠杆接近尾声
- 监管政策边际松动信号（如降低印花税讨论、IPO注册制推进）
- 机构对券商板块持仓处于历史低位

**顶部信号（卖出区域）**：
- 散户开户数激增，新基金发行火爆
- 市场日均成交量连续创历史新高
- 券商自营盘大幅扩张，股票仓位处于高位
- 行业并购整合预期炒作（往往出现在景气顶部）
- P/B超过2x，且分析师一致看多

---

### 证券股估值方法

#### 核心方法：P/B估值 + ROE弹性分析

**穿越周期ROE**（类似工业股的穿越周期P/E）：
```
穿越周期ROE = 近一个完整牛熊周期（约5~7年）的平均ROE
合理P/B = 穿越周期ROE / COE（股权成本，通常10~12%）

示例：穿越周期ROE = 10%，COE = 11%
→ 合理P/B ≈ 0.9x
→ 当前P/B < 0.8x → 可能低估
```

**景气弹性模型**（牛市情景下的盈利测算）：
```
经纪收入 = 日均成交量（ADT）× 佣金率 × 交易日数
投行收入 = 市场融资规模 × 市场份额 × 承销费率
自营收入 = 自营资产规模 × 预期收益率（股票/债券分开）
```

ADT对券商盈利的弹性极大，通常：ADT翻倍 → 龙头券商净利润增长60~80%

#### 辅助方法：分部估值（Sum of Parts）

对于业务多元化的大型券商：
- 经纪业务：按稳态收入 × 行业市销率
- 投行业务：按市场份额 × 行业估值
- 资管业务：AUM × 行业P/AUM倍数
- 自营资产：股票按市值，债券按账面价值，风险资产折价

---

### 证券股的监管维度

中国证券行业监管对估值的影响不亚于市场周期，必须单独评估：

**影响供给侧的监管**（决定竞争格局）：
- 券商牌照审批（新设/合并）
- 业务资格许可（衍生品、科创板跟投等）
- 净资本监管要求（影响杠杆上限）

**影响需求侧的监管**（决定行业蛋糕大小）：
- IPO节奏（注册制推进 vs 阶段性收紧）
- 印花税税率
- 两融规模及比例限制
- QFII/沪深港通开放程度

**监管周期与市场周期的叠加效应**：
- 监管宽松 + 市场上行 = 最强景气（双重催化）
- 监管收紧 + 市场下行 = 最深低谷（双重压制）
- 监管宽松 + 市场下行 = 政策底，往往是布局时机

## 三、保险公司

保险公司分寿险和财险，两者的分析框架差异极大，切勿混用。

### 寿险公司

#### 核心价值驱动

寿险的价值来源于**长期负债与长期资产之间的经营**，核心指标体系：

| 指标 | 含义 | 理想状态 |
|------|------|---------|
| **内含价值（EV）** | 现有业务未来现金流的折现值（存量价值） | 持续稳定增长 |
| **一年新业务价值（NBV）** | 当年新销售保单的折现价值（增量价值） | 增速 >10% |
| **NBV Margin** | NBV / 首年保费 | >30% 为优质 |
| **新业务价值增速** | YoY NBV增速 | 核心成长指标 |
| **投资收益率** | 资产端回报 | >5% 为健康 |

**寿险公司估值**：
```
P/EV（内含价值倍数）= 市值 / 内含价值
合理范围：0.8~1.5x（成熟市场），中国龙头历史峰值可达2x以上

隐含新业务价值倍数 = (市值 - EV) / NBV
通常 8~15x 为合理区间（反映对未来新业务增长的定价）
```

#### 寿险的周期驱动

**利率是最关键的变量**（双重影响）：
- 负债端：利率下行 → 准备金计提增加（压低当期利润）
- 资产端：利率下行 → 再投资收益率下降（影响长期投资收益）
- 因此寿险公司是典型的**利率敏感型**企业，利率上行环境显著利好

**代理人规模与产能**：
- 代理人数量（增员/脱落率）
- 人均产能（NBV / 代理人）
- 产能提升往往比规模扩张更可持续

#### 寿险周期信号

**底部信号**：
- P/EV跌破0.5x（历史极端低估区域）
- NBV连续两年负增长后触底
- 代理人队伍清虚接近尾声（低产能人员出清完毕）
- 利率见底预期升温

**顶部信号**：
- P/EV超过1.5x
- 代理人激进增员，人均产能下滑
- 短期储蓄险大量销售（高度依赖利差，利润质量差）

---

### 财险公司

财险的商业模式与工业企业更接近，可视为"收取保费、赔付风险"的周期性服务业。

#### 核心指标：综合成本率（COR）

```
综合成本率（COR）= 赔付率 + 费用率
COR < 100% → 承保盈利
COR > 100% → 承保亏损（需靠投资收益弥补）
```

优质财险公司的标志：**长期COR < 97%**（意味着承保本身持续盈利，投资收益是额外回报）

#### 财险估值方法

| 方法 | 说明 |
|------|------|
| P/B | 1~2x，配合ROE判断 |
| P/承保利润 | 排除投资收益波动 |
| 股息率 | 优质财险高分红，3~5% |

#### 财险的周期性

财险遵循**承保周期**（Underwriting Cycle）：
- 低赔付期 → 保费竞争激烈 → COR恶化 → 部分公司退出 → 保费价格回升 → 承保盈利改善
- 中国车险自2020年综合改革后，竞争格局显著向龙头集中，头部公司的周期波动趋于温和

## 四、商业银行

银行已在`multiples-guide.md`中有基础覆盖，此处补充周期分析维度。

### 信贷周期是核心驱动

```
经济扩张 → 贷款需求上升 → NIM扩张 → 资产质量好 → ROE提升 → P/B上行
经济收缩 → 贷款需求下降 → NIM收窄 → 不良率上升 → ROE下降 → P/B下行
```

### 银行股周期定位信号

**底部信号**：
- 不良贷款率见顶回落（或拨备覆盖率触底后回升）
- P/B跌至0.4~0.6x（历史底部区域，尤其股份制银行）
- 净息差（NIM）收窄触底，加息周期启动
- 房地产坏账充分出清的政策信号

**顶部信号**：
- 信贷高速扩张，银行主动降低信贷标准
- P/B超过1.2x（中国大型银行历史高位）
- 居民/企业杠杆率快速上升

### 银行特有的估值调整

**拨备前利润（PPOP）**比净利润更真实：
```
PPOP = 净利润 + 信用减值损失
PPOP增速排除了拨备政策变化的扰动，反映真实经营能力
```

**有形账面价值（TBV）**：
```
TBV = 净资产 - 商誉 - 无形资产
P/TBV更保守，在资产质量存疑时应优先使用
```
