# 估值的艺术 — 分析框架

本技能以《估值的艺术：110个解读案例》（Nicolas Schmidlin）为方法论基础，将书中框架应用于真实公司分析。

## 输出风格判断

- **用户上传财报 / 给出具体公司数据** → 结构化分析报告（见下方五步框架）
- **用户问"XX方法怎么用"、"PE和EV/EBIT有什么区别"等方法论问题** → 对话式解释，配合具体案例类比
- **混合问题（既有数据又有方法论困惑）** → 先做分析，分析过程中自然解释所用方法

---

## 五步分析框架

### Step 1：财务报表解读（定量基础）

**盈利能力**
- 收入增速（近3~5年CAGR，区分有机增长与并购驱动）
- 毛利率 / EBIT率 / 净利率趋势（关注压缩或扩张的原因）
- ROE / ROIC（优先ROIC，排除杠杆影响；ROIC > WACC是创造价值的核心信号）

**资产质量与资本结构**
- 净债务 / EBITDA（<2x为健康；>4x需警惕）
- 自由现金流（FCF）与净利润的匹配度（持续背离是财务质量警报）
- 营运资本变化（应收账款增速 > 收入增速 → 收入质量存疑）

**关键财务质量检查（书中强调）**
- 非经常性损益是否频繁出现（"一次性"亏损反复发生 = 经营性问题）
- 折旧政策与资本开支的关系（Capex持续 < 折旧 → 资产在萎缩）
- 现金转化率 = FCF / 净利润（>80%为优质）

---

### Step 2：商业模式分析（定性护城河）

依据书中框架，评估以下护城河来源：

| 护城河类型 | 判断信号 |
|-----------|---------|
| 成本优势 | 同行中毛利率领先 + 规模效应明显 |
| 品牌定价权 | 能持续提价而不失客户 |
| 网络效应 | 用户越多产品越有价值（平台类） |
| 转换成本 | 客户粘性高、续约率高、迁移成本大 |
| 特许经营权 | 牌照、专利、监管壁垒 |

**商业模式风险信号**：客户集中度过高（单客户>20%收入）、技术颠覆敞口、监管不确定性

---

### Step 3：估值方法矩阵

书中核心原则：**单一估值方法不可靠，交叉验证才有效**。

#### 相对估值（倍数法）

| 指标 | 适用场景 | 注意事项 |
|------|---------|---------|
| P/E | 盈利稳定的成熟企业 | 非经常性损益会扭曲 |
| EV/EBIT | 资本结构不同的公司横向对比 | 优于P/E，排除杠杆差异 |
| EV/EBITDA | 重资产行业、并购定价 | 忽略资本开支差异，重资产慎用 |
| P/B | 银行、保险、资产驱动型企业 | ROE需同步考察 |
| P/FCF | 现金流健康的轻资产企业 | 最难操控的估值指标 |
| EV/Sales | 亏损成长股、收入可见性高 | 需结合利润率改善预期 |

**可比公司分析要点**：选同行时注意增速、利润率、资本结构尽量接近；倍数折价/溢价需有明确理由

> ⚠️ **周期股特别提示**：若分析对象是钢铁、有色、化工、航运、半导体、能源等周期性行业，**在进入标准五步框架之前，必须先读 `references/cyclical-investing.md`**，完成周期定位和资产负债表强度判断，再做估值。单年P/E在周期股上几乎必然误导。

> ⚠️ **金融公司特别提示**：若分析对象是证券公司、经纪商、信托、资管公司，在使用 P/B-ROE 公式前，必须确认：（1）ROE 已完成多周期正规化（建议至少10年数据，含完整牛熊）；（2）参照 `references/multiples-guide.md` 金融（证券）专节，改用 **FCFE DCF 为主方法**，P/B-ROE 仅作交叉验证。

#### 绝对估值（DCF）

核心三要素：
1. **自由现金流预测**：前3年细致预测，后续用终值增长率（通常2~3%，不超过GDP增速）
2. **折现率（WACC）**：通常8~12%，高风险/新兴市场上浮
3. **终值敏感性**：终值往往占DCF总价值60~70%，必须做敏感性分析

> DCF的价值在于迫使你明确假设，而不是得出一个精确数字。

**Ke / WACC 参数校验（金融公司必做，其他公司推荐）**

设定折现率后，用历史价格反推隐含 Ke 进行交叉验证：

```
Ke_implied = g + (ROE_normalized - g) / PB_historical
```

对历史各年份计算该值，观察其分布。**校验目的是发现假设偏差，不是把 Ke 调低到"复现当前股价"**——后者是循环论证。

若 CAPM 估算值与历史隐含 Ke 均值相差超过 200bp，需排查：
- MRP 数据来源是否使用了当年 Damodaran 最新数据（见 `references/multiples-guide.md` 折现率参数数据源）
- Beta 是否参照了同行业 unlevered beta 后重新加杠杆
- BPS 是否低估了账外特许权价值（牌照、客户资产等），导致估值地板系统性偏低

#### 资产重置价值 / 清算价值
- 适用于控股公司、地产、资源类企业
- 关注隐蔽资产（低值摊销的土地、品牌）与隐性负债（养老金、或有负债）

---

### Step 4：综合估值判断

整合多种方法后，给出**估值区间**而非单点：

```
保守情景（清算/悲观DCF）：XXX
基准情景（主要倍数法 + 基准DCF）：XXX
乐观情景（高成长假设 + 溢价倍数）：XXX
```

**安全边际原则**（书中价值投资核心）：
- 当前价格 vs. 基准估值的折价幅度
- 一般要求 >20~30% 的安全边际才构成吸引力
- 对于高确定性、强护城河企业，可接受更低安全边际

---

### Step 5：投资结论与风险提示

输出应包含：
- **一句话结论**：高估 / 合理 / 低估，及置信度（高/中/低）
- **核心投资逻辑**（支撑低估判断的2~3个关键因素）
- **主要风险**（颠覆这一判断需要发生什么）
- **需要进一步核实的信息**（若财报信息不完整）

## 方法论解释原则

当用户询问估值方法时：
1. 先用一句话说清楚这个指标**衡量什么**
2. 举一个具体行业或公司类型的例子
3. 说明**什么时候这个方法会失效**（书中110个案例的核心洞见之一）
4. 如果有更适合的替代方法，主动建议

---

## 读取PDF财报的顺序

1. 先看**管理层讨论与分析（MD&A）**：了解公司自己的业务叙事
2. 再看**利润表**：营收 → 毛利 → 经营利润 → 净利润的逐级变化
3. 看**现金流量表**：经营活动现金流与净利润的差异
4. 看**资产负债表**：债务结构、商誉、无形资产
5. 重点关注**附注**：会计政策变更、关联方交易、或有负债

如果是港股/A股财报，额外注意：
- A股：非经常性损益单独披露，是财务质量的重要参考
- 港股：注意以人民币列报但以港币计价的外汇敞口
