# 周期性公司估值
## 来源：《Investment Valuation》第3版 Ch.13 + 《The Dark Side of Valuation》Ch.11 + 《学会估值轻松投资》Ch.10

---

## 核心挑战

周期性公司（钢铁、化工、航运、汽车、房地产、半导体、MDI/聚氨酯等）：
- **当期盈利严重失真**：繁荣期盈利虚高，衰退期盈利骤降甚至亏损
- **用当期盈利做倍数估值会系统性犯错**（繁荣期低估、衰退期高估）
- **周期预测困难**：没人能可靠预测周期拐点
- **经营杠杆放大波动**：固定成本高→收入小波动→利润大波动
- **融资成本随周期变化**：繁荣期利率低/风险溢价低，衰退期反之
- **再投资率随周期扭曲**：繁荣期CapEx膨胀（产能扩张冲动），底部期CapEx冻结

Damodaran 核心观点（第3版第13章原文）：

> "为进行正确的正规化处理，我们必须将原理落实到极致。除了对收益率进行正规化处理之外，资本收益率、再投资率和融资成本同样需要正规化。"

> "如果我们决定以正规化收益率替代这家公司的当期收益率，并且我们使用外部的收益增长率估计值来预测未来收益，那么我们必然会高估这些收入和公司价值。这实际上是对增长率进行了重复计算——首先反映在收入的正规化过程中，而后又体现为增长率的提高。"

## 方法一：全参数正规化 DCF（主方法，强制执行）

### 步骤0：判断公司所处周期位置

在开始正规化之前，必须先判断当前处于周期的哪个阶段：

| 信号 | 繁荣顶部迹象 | 衰退底部迹象 | 中性/正常 |
|------|------------|------------|----------|
| 产能利用率 | >90% | <70% | 75-85% |
| 库存周期 | 主动补库，库存高 | 被动去库，库存低 | 正常周转 |
| 行业 ROE | 远高于历史均值 | 远低于历史均值，甚至为负 | 接近历史均值 |
| 新增产能 | 大量在建项目 | 资本开支冻结 | 稳定 |
| 行业并购 | 高溢价收购 | 破产重组 | 正常 |
| 产品价格 | 接近历史高位 | 接近历史低位 | 中位数附近 |
| 当前利润率 | 远高于正常化利润率 | 远低于/为负正常化利润率 | 接近正常化 |

**输出格式（强制）：**
```
周期位置判断：[繁荣顶部 / 衰退底部 / 中性偏上 / 中性偏下]
判断依据：[具体列出3-5个信号指标的实际数值]
正规化必要性：[高/中/低]
```

⚠️ 如果判断为"繁荣顶部"或"衰退底部"，正规化是**强制性的**，不可跳过。即使在"中性"位置，也建议进行正规化并比较差异。

---

### 步骤1：正规化基期盈利

选择**正常化基础**（按可获取数据优先级排序）：

**选项A（最佳）：完整周期平均利润率（Damodaran推荐）**
- 取过去一个完整周期（通常7–15年）的**平均 EBIT margin**（不是平均EBIT绝对值）
- 用该平均 margin × 当期收入 = 正常化 EBIT
- 丰田案例：1998-2009年平均税前营业利润率7.33% × 当期收入 = 正常化营业利润

**选项B：相对平均值（推荐用于规模变化大的公司）**
- 对增长型周期公司，绝对平均EBIT会低估（因为公司规模已大幅增长）
- 改用：**平均利润率** × 当期收入（而非平均EBIT绝对值）
- 同理：用CapEx/Revenue比率的平均值（而非CapEx绝对值）

**选项C：行业均值**
- 用同行业可比公司的跨周期平均利润率
- 适用于历史数据不完整的公司
- 缺陷：无法体现公司相对同行的效率差异

**选项D：峰谷平均**
- 简单取最近一个周期高点和低点利润率的平均值
- 粗糙但快速，适用于初步判断

**Damodaran的警告（第3版第13章原文）：**

> "分析师往往会使用历史数据，并假设对过去一定时期内（如5-10年）的数据进行平均即可得到正规化结果。但得到的结果既有可能是正规化数值，也有可能不是，尤其是对大宗商品企业，考虑到每个价格周期可能延续数十年，因此，你得到的所谓正规化数值很可能是对真实价格的歪曲。这在2014年和2015年表现得一览无遗，当时大多数大宗商品的价格出现了下降。很多分析师和公司都认为，价格迟早会回归历史平均水平，基准就是大宗商品在2003-2013年的平均价格，但值得注意的是，在这段时期，中国进行了史无前例的基础设施投资，并由此带动大宗商品的价格一路创下历史新高。"

**因此：正规化期间应覆盖至少一个完整周期，且需检验该期间是否代表"正常"状态。**

### 步骤2：正规化再投资率（关键步骤，现有框架常遗漏）

**为什么必须正规化再投资率？**

- 周期高峰期：公司因乐观预期而大幅增加CapEx（扩产冲动）→ 再投资率虚高
- 周期低谷期：公司冻结CapEx（求生模式）→ 再投资率虚低
- 用当期CapEx估算再投资率，会导致FCFF系统性偏差

**正规化方法：**

```
正常化再投资率 = 正常化Net Investment / 正常化EBIT(1-t)

正常化Net Investment的计算：
方法1：用正常化CapEx比率
  正常化CapEx = 当期收入 × (历史平均CapEx/Revenue比率)
  正常化Net Investment = 正常化CapEx - D&A + 正常化ΔNWC

方法2：用基本面反推
  正常化再投资率 = 目标增长率 / 正常化ROIC
  其中正常化ROIC = 正常化EBIT(1-t) / 正常化Invested Capital
```

**与增长率的一致性约束（Damodaran强调）：**

$$g = ROIC_{normalized} \times Reinvestment\ Rate_{normalized}$$

如果公司处于稳定增长阶段，增长率为GDP增速（约5%名义），那么：
- 正常化ROIC = 12%
- 则正常化再投资率 = 5% / 12% = 41.7%

⚠️ **禁止：用正规化EBIT但搭配非正规化CapEx——这会导致FCFF计算不一致。**

---

### 步骤3：正规化WACC（关键步骤，现有框架常遗漏）

**为什么必须正规化WACC？**

Damodaran第3版第13章原文：

> "对周期型企业或大宗商品企业，将再投资及资本成本的取值锁定在最近一期的收益上，是造成估值错误的不二法则。"

周期对WACC各组成部分的影响：

| WACC组成部分 | 繁荣期影响 | 衰退期影响 | 正常化处理 |
|-------------|-----------|-----------|-----------|
| 无风险利率 | 可能被央行调高 | 被央行调低 | 用中期均值或当前值 |
| 股权风险溢价(ERP) | 低（风险偏好强） | 高（风险厌恶） | 用Damodaran隐含ERP |
| Beta | 偏低（顺周期） | 偏高（逆周期） | 用行业Unlevered Beta重新加杠杆 |
| 债务成本 | 低（违约利差小） | 高（违约利差大） | 用正常化信用评级 |
| 债务比率 | 偏高（借债扩产） | 偏低（去杠杆） | 用目标/行业平均资本结构 |

**正规化方法：**

```
正规化WACC计算：
1. Beta：使用行业Unlevered Beta + 正常化D/E比率
   （不用当期回归Beta，也不用当期市值D/E——后者在周期底部会被严重扭曲）
2. ERP：使用Damodaran当年更新的隐含ERP
3. 债务成本：基于正常化利息覆盖率 → 合成评级 → 违约利差
4. 债务比率：使用行业平均或公司长期平均（不用当期市值比率）
```

---

### 步骤4：增长率去重检查（关键步骤，现有框架常遗漏）

Damodaran第3版第13章原文：

> "如果我们决定以正规化收益率替代这家公司的当期收益率，并且我们使用外部的收益增长率估计值来预测未来收益，那么我们必然会高估这些收入和公司价值。这实际上是对增长率进行了重复计算——首先反映在收入的正规化过程中，而后又体现为增长率的提高。"

**核心规则：正规化盈利 + 外部高增长率 = 重复计算**

**检查方法：**
```
如果基期盈利已正规化（剔除了周期低谷的影响）：
  → 预测期增长率不应包含"从低谷恢复"的成分
  → 增长率应 = 行业长期增速 或 基本面增长率（g = ROIC × RR）
  → 禁止使用分析师一致预期增长率（其中包含周期恢复因素）

如果使用当期盈利（低谷期）：
  → 增长率可以包含周期恢复
  → 但终值必须用正常化盈利（不能假设永远在低谷或永远在高峰）
```

**实操示例：**
```
假设：万华化学当期（衰退期）EBIT = 80亿，正常化EBIT = 250亿
错误做法：正常化EBIT=250亿 + 增长率=15%（含周期恢复）→ 严重高估
正确做法A：正常化EBIT=250亿 + 增长率=5%（行业长期增速）→ 合理
正确做法B：当期EBIT=80亿 + 增长率=25%（前3年恢复期）→ 正常化后再降速 → 合理但复杂
```

---

### 步骤5：一致性自检（最终检查，不可跳过）

正规化DCF完成后，必须通过以下检查：

```
全参数正规化一致性检查清单：
[ ] 正常化EBIT = 正常化margin × 当期收入（或历史平均margin × 当期收入）
[ ] 正常化CapEx = 正常化CapEx/Revenue × 当期收入
[ ] 正常化WACC = 正常化Beta + 正常化ERP + 正常化债务成本 + 正常化D/E
[ ] g = ROIC_normalized × RR_normalized（增长率与再投资率一致）
[ ] 终值增长率 ≤ 名义GDP增速
[ ] 预测期增长率不含"从周期恢复"的重复计算
[ ] 终值占比 ≤ 80%（如果>80%，检查永续增长假设是否合理）
```

⚠️ **任何一项不通过，必须修正后重新计算。**

---

### 完整操作流程图

```
[步骤0] 判断周期位置
    ↓
[步骤1] 正规化基期EBIT（历史平均margin × 当期收入）
    ↓
[步骤2] 正规化再投资率（正常化CapEx比率，或基本面反推）
    ↓
[步骤3] 正规化WACC（行业Beta + 隐含ERP + 正常化信用评级）
    ↓
[步骤4] 增长率去重检查（正规化后不用周期恢复增长率）
    ↓
[步骤5] 一致性自检（g = ROIC × RR, 终值占比<80%）
    ↓
[输出] 正规化DCF → 每股内在价值
```

## 方法二：概率估值法（Damodaran第3版第3章+第13章推荐）

**为什么周期性公司特别适合概率估值？**

Damodaran第3版第13章原文：

> "由于周期型及大宗商品企业的收益、现金流和价值在很大程度上取决于少数宏观经济变量，因此，概率方法尤其适用于这类公司。"

> "最适合采用概率估值法的环境，是公司的基本价值仅依赖于一个或两个变量，而且有足够历史信息可用来对概率分布及参数做出估计。"

### 方法A：离散情景分析（推荐，实操性强）

```
1. 识别关键变量（如：MDI价格、GDP增速、产能利用率）
2. 设定3-5个情景：

   情景          概率     关键假设              DCF结果
   ────────────────────────────────────────────────
   繁荣延长      15%      产品价格持续高位       V₁
   基准回归      50%      价格回归正常化水平     V₂
   深度衰退      25%      价格跌破成本线         V₃
   结构性恶化    10%      需求永久性下滑         V₄

3. 加权预期价值 = Σ(Pᵢ × Vᵢ)
4. 输出：期望价值 + 价值区间 [V₃, V₁] + VaR（最坏5%情景的价值）
```

**概率分配原则：**
- 不应均匀分布——参考历史频率（过去30年中，各情景实际发生了多少次？）
- 如果公司有"反脆弱"特征（低谷期能活得比别人久），底部情景概率可降低
- 宏观环境（货币政策、财政政策）应影响概率分配

### 方法B：蒙特卡洛模拟（适用于数据充足的情况）

```
1. 确定关键输入变量的概率分布：
   - 产品价格：对数正态分布（参数从历史数据估计）
   - 产量/销量：正态分布
   - 成本：正态分布

2. 建立映射：价格 → EBIT → FCFF → DCF → 每股价值

3. 运行10000次模拟

4. 输出：
   - 期望每股价值
   - 价值分布（标准差、偏度）
   - VaR（5%/95%分位数）
   - 低于当前股价的概率
```

**埃克森美孚案例（Damodaran第3版第13章）：**
- 10000次模拟，石油价格为随机变量
- 结果：平均每股价值69.59美元，最低2.25美元，最高324.42美元
- 低于当时股价64.83美元的概率 > 50%
- 结论：在当前油价下，估值结果高度不确定，取决于油价走势判断

⚠️ **概率估值法不替代正规化DCF，而是对它的补充。每个情景下的DCF仍须满足基本一致性约束。**

## 方法三：正常化倍数估值（替代/验证方法）

用**跨周期 P/E（Normalized P/E）**：

$$Normalized\ P/E = \frac{Current\ Price}{Average\ EPS\ over\ cycle}$$

也称为 **Shiller P/E（CAPE）** 思路，在个股层面的应用。

Damodaran第3版第13章原文对正常化倍数的论述：

> "如果要让周期性或大宗商品企业的正规化收益能反映其正常年份的经营成果，那么市场对公司的估值方式就必须与这些正规化收益保持一致。"

> "和收益更稳定的企业相比，我们应预期的是收益风险更大的公司，其交易价格对正规化收益的倍数更低。"

**因此，正常化P/E不能简单跨公司比较，必须控制：**
- 增长率差异（高增长公司 → 更高正常化P/E）
- 风险差异（高Beta/高经营杠杆 → 更低正常化P/E）

| 指标 | 衰退期误读 | 正确做法 |
|-----|---------|---------|
| P/E | 分母小 → P/E 极高，错误显示"贵" | 用10年平均EPS |
| EV/EBITDA | 分母小 → 倍数高 | 用正常化 EBITDA |
| P/B | 净资产受减值影响 | 结合历史 P/B 区间 |
| 正常化P/E | 忽略增长率和风险差异 | 正常化P/E需经增长率和风险调整（回归法） |

## 方法四：资产重置价值法（资本密集型周期行业）

适用于钢铁、化工、矿业、航运：

$$Enterprise\ Value = Replacement\ Cost\ of\ Assets \times (1 + Premium/Discount)$$

- **Premium**：若 ROE > COE，公司创造价值，应溢价
- **Discount**：若 ROE < COE（周期底部常见），应折价
- 也称为 **Tobin's Q** 逻辑

---

## 方法五：商品价格情景法（适用于大宗商品下游/原材料公司）

当公司盈利主要取决于某一商品价格（如MDI价格、原油价格、铜价）时：

**步骤1：建立盈利-价格回归**
$$Operating\ Income = a + b \times Commodity\ Price$$

Damodaran用埃克森美孚1985-2008年数据回归：R²=90%，b=91.1亿美元/10美元油价

**步骤2：正常化价格 → 正常化盈利**
用正常化商品价格代入回归方程，得到正常化营业利润

**步骤3：多价格情景DCF**
```
情景          正常化价格    正常化EBIT    DCF每股价值
────────────────────────────────────────────────
悲观          当前价格-20%  低估值        V₃
基准          当前价格      中间值        V₂
乐观          当前价格+30%  高估值        V₁
```

## 常见错误警示

1. **繁荣期用当期 P/E 说"估值便宜"** → 实际是周期高点，未来盈利将大幅下滑
2. **衰退期因 P/E 极高或亏损放弃估值** → 恰恰可能是买入机会
3. **把短期利润率改善当作永久性改善** → 需区分周期性反弹 vs 结构性改善
4. **对自然资源公司用历史盈利正常化** → 还需对大宗商品价格本身正常化（见 `darkside-commodity.md`）
5. **正规化了EBIT但没有正规化再投资率和WACC** → 参数不一致，估值结果不可靠（**最常见的错误**）
6. **正规化EBIT后又用了包含"周期恢复"的增长率** → 重复计算，严重高估
7. **用最近5年平均利润率作为正规化值，但该5年本身不代表正常周期** → 2003-2013年大宗商品超级周期就是典型反例
8. **只做单一点估计DCF，不做情景分析** → 周期股的价值本质上是一个分布，不是一个点
