# 金融服务公司估值
## 来源：《Investment Valuation》第3版 Ch.21–22；《The Dark Side of Valuation》第2版 Ch.14；《学会估值，轻松投资》Ch.16–17

---

## 为什么金融股需要特殊处理

金融服务公司（商业银行、投资银行、保险公司、证券公司、消费金融、信托）与一般公司在财务结构上有根本性差异，使得标准 DCF 框架无法直接适用。

**《Investment Valuation》第3版第21章的核心对比**：

| 维度 | 一般公司 | 金融公司 |
|------|---------|---------|
| 债务性质 | 融资工具——借钱的目的是投资于运营 | **经营原材料**——存款/借款是"库存"，就像制造商的原材料 |
| 资本定义 | 投入运营的总资本（债务 + 股权） | **仅监管资本**（Tier 1 + Tier 2） |
| FCFF 定义 | 清晰（税后营业利润 - 再投资） | **模糊**——债务变化本身就是经营活动的一部分 |
| CapEx | 有形资产投资，清晰可分 | 难以与投资组合变化区分 |
| 营运资本 | 可计算、有意义 | **无意义**——银行没有传统意义的"存货"或"应收账款" |
| 杠杆率 | 管理层选择（资本结构决策） | **监管强制**（资本充足率约束） |
| 再投资 | 投资于工厂/设备/研发 | 贷款增长/保费增长/投资组合扩张 |
| 税盾 | 利息税盾明确 | 利息是收入的一部分，不是税盾 |

**核心结论**：

> 对金融公司不使用 FCFF DCF 和 WACC，而是使用 FCFE（股权自由现金流）折现模型或股息折现模型（DDM），仅在股权层面估值。折现率仅使用股权成本（Cost of Equity），不使用 WACC。

## 步骤0：金融公司子分类

不同类型的金融公司估值方法有显著差异：

| 类型 | 代表 | 主要估值方法 | 特殊指标 |
|------|------|-------------|---------|
| 商业银行 | 工商银行、招商银行 | FCFE DCF / P-B/ROE | NIM、NPL、CET1 |
| 投资银行 | 高盛、中信证券 | FCFE DCF / P-E | ROA、杠杆率、收入结构 |
| 寿险公司 | 中国平安、中国人寿 | EV 法（P/EV） | EV、VNB、NB Margin |
| 财险公司 | 人保财险、PICC | P/B + 综合成本率 | COR、承保利润 |
| 消费金融/ fintech | 蚂蚁集团、LendingClub | FCFE DCF / P-E | NIM、坏账率、获客成本 |
| 证券/资管 | 东方证券、贝莱德 | FCFE DCF / AUM 倍数 | AUM、管理费率 |

**判断规则**：根据公司类型选择对应方法，不要用银行的方法去估保险公司，反之亦然。

## 方法一：FCFE DCF（股权自由现金流折现）

### 适用范围
所有类型的金融公司，尤其是非寿险公司、银行、投行、fintech。

### FCFE 的计算逻辑

**《Investment Valuation》第3版第22章的框架**：

金融公司的 FCFE 与一般公司完全不同。金融公司的"再投资"不是建工厂买设备，而是**资本增长（贷款增长/保费增长/投资组合扩张）**所需的监管资本增量。

$$FCFE = Net\ Income - \Delta Required\ Equity\ Capital$$

其中：

$$\Delta Required\ Equity\ Capital = Target\ CET1\ Ratio \times \Delta RWA$$

- **Net Income**：净利润（税后）
- **Target CET1 Ratio**：目标核心一级资本充足率（如 11–13%）
- **ΔRWA**：风险加权资产增量

**具体到银行的操作步骤**：

1. **预测贷款/资产增长**：
   - 基于宏观经济假设（GDP 增长率 + 信贷增速）
   - 区分生息资产（贷款、债券投资）和非生息资产（现金、固定资产）

2. **预测净息差（NIM）**：
   - 资产端收益率 - 负债端成本率
   - 考虑利率周期对 NIM 的影响
   - 长期稳态 NIM 参考历史均值和同行

3. **预测拨备前利润（PPOP）**：
   $$PPOP = Net\ Interest\ Income + Non\text{-}interest\ Income - Operating\ Expenses$$
   - PPOP 比 Net Income 更稳定，能反映真实经营能力

4. **预测信用成本（Credit Cost）**：
   $$Credit\ Cost = Expected\ NPL\ Ratio \times Loan\ Loss\ Provision\ Rate$$
   - 参考历史平均不良率 + 宏观周期判断
   - 关注关注类贷款迁徙率

5. **计算净利润**：
   $$Net\ Income = (PPOP - Credit\ Cost) \times (1 - Effective\ Tax\ Rate)$$

6. **计算 FCFE**：
   $$FCFE = Net\ Income - CET1\ Target \times \Delta RWA$$

### 折现率

$$k_e = R_f + \beta_{financials} \times ERP$$

**Beta 的选择**：
- 使用**金融业 Beta**，非全市场 Beta
- 银行 Beta 通常 0.8–1.3（取决于利率敏感度和杠杆水平）
- 保险 Beta 通常 0.7–1.1（寿险偏低，财险偏高）
- 证券 Beta 通常 1.2–1.8（与市场波动高度相关）

**无风险利率的选择**：
- 使用10年期国债收益率
- 如果公司在国内运营，使用本币国债收益率
- 对于跨国金融公司，使用加权平均

**⚠️ 不使用 WACC**：金融公司的"负债"（存款、借款）是经营性负债，不是融资性负债，不应在 WACC 中处理。

### 三阶段 FCFE 模型

**阶段1：高增长期（Year 1–5）**
- 贷款/资产增速高于行业平均
- NIM 逐步改善或维持高位
- FCFE 可能为负（高增长需要留存更多资本）
- k_e 较高（反映增长不确定性）

**阶段2：过渡期（Year 6–10）**
- 增速向行业平均收敛
- NIM 向稳态收敛
- FCFE 转正
- k_e 向行业均值收敛

**阶段3：稳态期（Terminal）**
- 贷款增速 = 名义 GDP 增长率 ± 1pp
- NIM = 长期历史均值
- ROE = k_e（或略高，反映特许权价值）
- g ≤ 名义 GDP 增长率
- 终值 = FCFE_terminal / (k_e - g)

## 方法二：股息折现模型（DDM）

### 适用范围
股息政策**稳定且可预测**的成熟银行/保险公司。

### 基础模型

$$P_0 = \sum_{t=1}^{n} \frac{DPS_t}{(1+k_e)^t} + \frac{P_n}{(1+k_e)^n}$$

### Gordon 增长模型（永续增长简化版）

$$P_0 = \frac{DPS_1}{k_e - g}$$

### 可持续增长率的推导

$$g = ROE \times Retention\ Ratio = ROE \times (1 - Payout\ Ratio)$$

**因此**：

$$P_0 = \frac{DPS_1}{k_e - ROE \times (1 - Payout\ Ratio)} = \frac{ROE \times (1 - b) \times BV_0}{k_e - ROE \times b}$$

其中 b = Retention Ratio。

### DDM 的适用前提

1. **派息率稳定且可持续**——如果公司频繁调整股息率，DDM 不适用
2. **增长率合理**——g 必须小于 k_e，且不超过名义 GDP 增长率
3. **公司不会因为监管要求突然削减股息**

**《Investment Valuation》第3版第22章**："DDM 对银行估值看似优雅，但依赖两个脆弱的假设：公司会持续派息、派息率不会因监管变化而大幅调整。在监管环境多变的市场（如中国），DDM 的可靠性低于 FCFE DCF。"

### DDM vs FCFE DCF

| 维度 | DDM | FCFE DCF |
|------|-----|---------|
| 适用对象 | 股息稳定的成熟银行 | 所有金融公司 |
| 对股息政策的依赖 | 高 | 低 |
| 对管理层灵活性的反映 | 差 | 好 |
| 计算复杂度 | 低 | 中等 |
| 中国银行适用性 | 有限（股息政策受监管影响大） | 更适用 |

**建议**：优先使用 FCFE DCF，DDM 作为交叉验证。

## 方法三：P/B 与 ROE 联动（最常用实践方法）

### 核心公式（Damodaran 推导）

$$\frac{P}{B} = \frac{ROE - g}{k_e - g}$$

**推导过程**（从 DDM 出发）：

$$P_0 = \frac{DPS_1}{k_e - g} = \frac{ROE \times BV_0 \times Payout}{k_e - g} = \frac{ROE \times BV_0 \times (1 - g/ROE)}{k_e - g} = BV_0 \times \frac{ROE - g}{k_e - g}$$

两边除以 BV_0：

$$\frac{P}{B} = \frac{ROE - g}{k_e - g}$$

### 关键推论

| 条件 | P/B | 经济含义 |
|------|-----|---------|
| ROE = k_e | P/B = 1.0 | 股权回报恰好补偿投资者要求，价值 = 账面 |
| ROE > k_e | P/B > 1.0 | 创造经济利润，应溢价 |
| ROE < k_e | P/B < 1.0 | 破坏经济利润，应折价 |
| ROE >> k_e | P/B >> 1.0 | 强大特许权价值（但需验证持续性） |
| g → k_e | P/B → ∞ | 分母趋零（此时公式失效，说明 g 假设过高） |

### 实操步骤

1. **估算银行的可持续 ROE**：
   - 用 ROAE（净利润 / 平均股东权益）
   - 预测期 ROE 考虑：NIM 趋势、信用成本周期、杠杆约束
   - 长期稳态 ROE 参考：成熟市场银行 8–12%，中国大型银行 10–14%

2. **确定合理 g**：
   - g ≤ 名义 GDP 增长率（通常 2–5%）
   - g = ROE × Retention Ratio（必须一致）

3. **估算 k_e**：
   - k_e = R_f + β × ERP

4. **计算理论 P/B**：

   **示例**：ROE = 12%，k_e = 10%，g = 4%
   $$P/B = (12\% - 4\%) / (10\% - 4\%) = 8\% / 6\% = 1.33\times$$

5. **与当前 P/B 对比**：
   - 当前 P/B < 理论 P/B → 被低估
   - 当前 P/B > 理论 P/B → 被高估（或市场预期更高 ROE）

### 截面回归法（同行对比增强版）

$$P/B = a + b \times ROE + c \times Growth + d \times Risk + \epsilon$$

- 用同行业上市公司的截面数据做回归
- R² 通常 >0.5，说明 ROE 对 P/B 有很强的解释力
- 识别异常值：P/B 远高于或低于回归预测的公司

**《学会估值》第16章**："P/B 是银行估值最可靠的锚点，但必须与 ROE 配合使用。单独看 P/B 是 0.5 还是 1.5 没有意义——0.5×P/B 的银行如果 ROE 只有 5%（< k_e），可能仍然不便宜。"

## 方法四：超额收益模型（Excess Return Model / EVA）

### 框架

$$Value\ of\ Equity = Book\ Value_0 + \sum_{t=1}^{\infty} \frac{(ROE_t - k_e) \times BV_{t-1}}{(1+k_e)^t}$$

**经济含义**：

- **第一项**：账面价值 = 如果公司只能赚取 k_e 的回报，股权值多少
- **第二项**：超额回报的现值 = 如果 ROE 持续高于 k_e，额外值多少

### 与 P/B-ROE 公式的关系

超额收益模型是 P/B-ROE 公式的动态版本：
- P/B-ROE 公式假设 ROE 立即达到稳态
- 超额收益模型允许 ROE 逐年变化，更灵活

### 适用场景

- ROE 正在从低水平恢复的公司（如银行不良出清后的 ROE 修复）
- 新银行/新设立的金融子公司（初期 ROE 低，逐步提升）

### 实操注意

- **必须预测 ROE 向稳态收敛的路径**——不能假设当前异常高/低的 ROE 永续
- **账面价值的增长**取决于留存收益，即 BV_t = BV_{t-1} + Net Income_t - DPS_t
- 如果预测期内 ROE < k_e，超额收益为负，价值低于账面值

## 方法五：剩余收益模型（Residual Income Model）

与超额收益模型本质相同，但更常用于保险公司的估值文献中。

$$P_0 = BV_0 + \sum_{t=1}^{T} \frac{RI_t}{(1+k_e)^t} + \frac{Terminal\ Value_T}{(1+k_e)^T}$$

$$RI_t = Net\ Income_t - k_e \times BV_{t-1} = (ROE_t - k_e) \times BV_{t-1}$$

**优势**：
- 不需要预测终值的增长率（终值可以是剩余收益的永续年金）
- 价值分解为已知量（BV_0）+ 未知量（未来超额收益的现值）
- 天然适用于账面价值占比较大的金融公司

## 寿险公司估值（P/EV 法）

### 为什么寿险不能用银行的方法

寿险公司的业务本质与银行完全不同：

| 维度 | 银行 | 寿险公司 |
|------|------|---------|
| 核心业务 | 吸收存款 → 发放贷款 | 收取保费 → 承担长期风险 |
| 负债性质 | 短期存款 | 长期准备金（10–30年） |
| 利润确认 | 当期利润基本准确 | 利润严重滞后（新业务利润分多年释放） |
| 关键指标 | NIM、NPL | EV、VNB、NB Margin |
| 价值来源 | 利差 × 贷款规模 | 新业务价值累积 + 投资偏差 |

**核心问题**：寿险公司的当期利润不能反映真实价值。一份保单在签订时就产生了大部分价值，但会计上分多年确认利润。因此需要**内含价值（Embedded Value）**来衡量真实价值。

### 内含价值（EV）的完整拆解

$$EV = ANV + VIF$$

**1. ANV（Adjustable Net Asset Value / 调整后净资产）**

$$ANV = Shareholders' Equity + Reserves - Goodwill + Embedded\ Value\ Adjustments$$

调整项包括：
- 将准备金从法定准备金调整为经济准备金（更贴近真实负债）
- 扣除未来法定利润税（因为法定利润不等于经济利润）
- 资本性投入的调整

**2. VIF（Value of In-force Business / 有效业务价值）**

$$VIF = \sum_{policy_i} PV(Expected\ Profits\ from\ Existing\ Policies)$$

- 对现有有效保单逐单计算未来预期利润的现值
- 折现率通常使用**风险调整后折现率**（而非 WACC）
- 不包含未来新业务产生的利润

### 完整估值公式

$$Value = EV + PV(Future\ VNB)$$

$$= ANV + VIF + PV(Future\ VNB)$$

- **EV（内含价值）**：反映已有业务的价值 = ANV + VIF
- **PV(Future VNB)**：反映未来新业务产生的价值
- **VNB（Value of New Business）**：一年新签保单的内含价值

### VNB 的计算

$$VNB = \sum_{new\_policies} PV(Expected\ Profits)$$

关键参数：
- **NB Margin（新业务利润率）**：VNB / 新业务保费
- 通常用 "VNB Margin" 做同行比较
- 成熟寿险公司 NB Margin 通常 15–30%

### P/EV 倍数的解读

$$P/EV = \frac{Market\ Cap}{EV}$$

| P/EV 区间 | 含义 |
|-----------|------|
| 0.5–0.8× | 市场认为公司无法有效销售新业务，或现有业务价值被高估 |
| 0.8–1.2× | 合理区间（隐含适度的未来新业务预期） |
| 1.2–1.5× | 市场对公司新业务增长有较高预期 |
| >1.5× | 极度乐观（需验证 NB 增速假设是否合理） |

**⚠️ P/EV 的陷阱**：
- 不同公司的 EV 计算方法可能不同（折现率假设、死亡率假设、费用假设）
- 比较前必须确认各公司的 EV 计算口径一致
- P/EV 不反映资产端的投资风险（如果投资组合大量配置高风险资产）

### 折现率假设（寿险特有）

寿险公司使用**多档折现率**，而非单一 WACC：

| 折现率用途 | 典型值 | 说明 |
|-----------|-------|------|
| VIF 折现 | 8–11%（视市场和公司） | 反映保单风险的股权折现率 |
| EV 折现 | 与 VIF 相同或略低 | 整体内含价值的折现 |
| PV(Future VNB) | 通常与 VIF 相同 | 新业务与现有业务风险相似 |
| 投资组合折现 | 按资产类别（债券 3–5%，股票 8–12%） | 反映不同资产的风险 |

**《学会估值》第17章**："寿险估值最大的错误是在 EV 计算中使用过低的折现率。如果把寿险的 EV 折现率设为 5%（像债券一样），会严重高估价值。寿险的负债具有股权风险特征，折现率应反映这一点。"

## 财险公司估值

### 核心指标

财险公司的估值比寿险简单，因为：
- 负债期限短（通常 1 年）
- 利润确认更及时
- 没有长期准备金的复杂估算

**核心指标**：

1. **综合成本率（Combined Ratio, COR）**：
   $$COR = \frac{Incurred\ Losses + Underwriting\ Expenses}{Earned\ Premiums}$$
   - COR < 100%：承保盈利（赚钱）
   - COR = 100%：承保盈亏平衡
   - COR > 100%：承保亏损（靠投资收益弥补）

2. **承保利润率**：100% - COR

3. **投资收益率**：保费收入产生的投资组合回报

### 财险估值方法

$$Value = ANV + \sum_{t=1}^{\infty} \frac{(Underwriting\ Profit_t + Investment\ Income_t - k_e \times BV_{t-1})}{(1+k_e)^t}$$

**简化版**：对于成熟财险公司，P/B 与 ROE 联动公式仍然适用：

$$P/B = \frac{ROE - g}{k_e - g}$$

但需注意：
- 财险 ROE 的波动性大于银行（受巨灾影响）
- 稳态 ROE 应使用 5–10 年平均（平滑巨灾年份）
- COR 的周期性是财险估值的关键挑战

## 银行特殊问题处理

### 1. 不良贷款（NPL）与拨备

**影响估值的关键变量**：

$$Credit\ Cost = NPL\ Ratio \times Provision\ Coverage \times Expected\ Loss\ Rate$$

**估值中必须做的调整**：
- 使用**预期信用损失**（IFRS 9），而非已发生损失
- 关注**关注类贷款迁徙率**（关注类 → 不良的比例）
- 考虑经济周期对不良率的影响（逆周期调整）
- 拨备覆盖率不是越高越好——过高拨备可能隐藏利润（"拨备池"概念）

**《Investment Valuation》第3版第22章**："银行拨备是利润最大的调节器。分析银行估值时，必须检查拨备假设是否一致且合理——如果一家银行的拨备覆盖率远高于同行，要么它真的更保守，要么它在隐藏利润。"

### 2. 表外业务与影子银行

中国银行业的特殊问题：
- 理财产品（表外资产）
- 非标投资
- 银承汇票、信用证
- 委托贷款

**估值处理**：
- 如果表外资产有明确的信用风险 → 应按风险权重计入 RWA
- 如果表外资产有隐性担保 → 应调整 FCFE（预期损失）
- 信息不透明的表外业务 → 在折现率中加入额外风险溢价

### 3. 利率风险

银行价值对利率高度敏感：
- 利率上升 → NIM 可能先升后降（资产端重定价快于负债端）
- 利率下降 → NIM 可能先降后升
- 久期错配 → 利率变动对净值的影响

**估值中应做利率敏感性分析**：
- 情景1：利率上升 100bp → 对 ROE 和 P/B 的影响
- 情景2：利率下降 100bp → 对 ROE 和 P/B 的影响
- 用敏感性分析来检验估值的稳健性

### 4. 息差管理

$$NIM = \frac{Net\ Interest\ Income}{Average\ Earning\ Assets}$$

**估值中的 NIM 预测**：
- 不要假设当前 NIM 永续
- 长期均衡 NIM 参考历史均值
- 考虑利率市场化对 NIM 的结构性压制

## 保险股特殊规则（用户定制）

基于用户投资组合包含中国平安等保险股，以下规则强制执行：

1. **寿险公司必须使用 P/EV 估值**，不可用 P/B 或 P/E 替代
2. **EV 必须使用寿险 EV**（非财险 EV），且需披露折现率假设
3. **VNB 是核心增长指标**——估值分析必须包含 NB Margin 和 VNB 增速
4. **ANV + VIF + PV(Future VNB) = 完整估值**，三部分缺一不可
5. **投资偏差（Experience Variance）**必须单独分析——投资收益偏差可以极大影响短期利润但不反映经营质量
6. **剩余边际（Residual Margin）摊销**是中国会计准则下的关键概念，需理解其对利润确认时间的影响

## 金融服务公司估值自检清单

### 通用检查

- [ ] 未使用 FCFF DCF 和 WACC（金融公司禁用）
- [ ] 折现率仅使用 k_e（股权成本）
- [ ] Beta 使用金融业 Beta，非全市场 Beta
- [ ] 未使用 EV/EBITDA 或 EV/EBIT（对金融公司无意义）
- [ ] 资本充足率假设符合监管要求（CET1 > 最低监管要求）

### 银行特有检查

- [ ] NIM 预测已考虑利率周期
- [ ] 信用成本预测基于预期信用损失（IFRS 9）
- [ ] 不良率和关注类迁徙率已分析
- [ ] 表外业务风险已评估并纳入 RWA
- [ ] P/B 与 ROE 的关系已验证（ROE > k_e → P/B > 1）
- [ ] 利率敏感性分析已完成

### 保险特有检查

- [ ] 寿险使用 P/EV，未用 P/B
- [ ] EV 计算的折现率假设已披露
- [ ] VNB 和 NB Margin 已分析
- [ ] ANV + VIF + PV(Future VNB) 三部分完整
- [ ] 投资偏差单独分析
- [ ] 剩余边际摊销对利润确认的影响已理解
- [ ] 财险 COR 使用 5–10 年平均（平滑巨灾影响）

### 一致性检查

- [ ] g = ROE × Retention Ratio（DDM/FCFE 中必须一致）
- [ ] 终值 P/B 不超过 3×（除非有极强的特许权证据）
- [ ] k_e - g > 0（否则模型发散）
- [ ] ROE 在终值年收敛到稳定值（通常 k_e ± 2pp）

## 常见错误警示

1. **对银行使用 FCFF/WACC**——金融公司没有传统意义的"债务"，FCFF 概念失效
2. **对银行使用 EV/EBITDA**——EV 对银行无意义（无法定义"企业价值"）
3. **寿险用 P/B 估值**——P/B 忽略了有效业务价值，严重低估寿险公司
4. **忽略资本充足率约束**——银行的增长受监管资本约束，不能假设无限增长
5. **P/B 与 ROE 分离解读**——低 P/B 可能是合理的（如果 ROE < k_e），不一定"便宜"
6. **拨备覆盖率越高越好**——过高拨备可能隐藏利润
7. **NIM 永续假设**——当前高 NIM 可能是利率周期的产物
8. **EV 计算口径不一致**——不同公司的折现率假设差异大，比较前必须统一口径
9. **忽略利率敏感性**——银行和保险公司的价值对利率高度敏感
10. **寿险当期利润 = 真实盈利**——寿险利润确认严重滞后，必须用 EV 体系
