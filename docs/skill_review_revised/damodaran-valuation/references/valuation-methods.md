# 第一部分：DCF 方法选型矩阵

## 三种 DCF 变体对比

| 维度 | FCFF DCF（企业自由现金流） | FCFE DCF（股权自由现金流） | DDM（股息折现） |
|------|------------------------|------------------------|--------------|
| 估值对象 | 企业整体价值 → 减债务 → 股权价值 | 直接估值股权价值 | 直接估值股权价值 |
| 折现率 | WACC | k_e（股权成本） | k_e（股权成本） |
| 适用范围 | 非金融公司（首选） | 金融公司 / 杠杆稳定公司 | 股息稳定的成熟公司 |
| 债务处理 | 在 WACC 中通过杠杆率体现 | FCFF 中已扣除利息，直接折现 | 利息已从利润中扣除 |
| 终值 | FCFF / (WACC - g) | FCFE / (k_e - g) | DPS / (k_e - g) |
| 中国A股适用性 | 最广 | 金融公司首选 | 有限（股息不稳定） |

## 选择流程

```
公司是否有大量经营性负债（存款/保费）？
├── 是 → 金融公司 → FCFE DCF 或 DDM（见 darkside-financials.md）
└── 否 → 非金融公司
         ├── 盈利为正 + 历史数据充足 → FCFF DCF（见 dcf-standard.md）+ Fair P/E 交叉验证
         ├── 盈利为负 / 高成长 → 调整后 DCF（见 darkside-growth.md）
         ├── 周期性公司 → Fair P/E（正常化EPS）为主 + 正常化 DCF 为辅（见 darkside-cyclical.md）
         ├── 大宗商品公司 → 价格情景 DCF + NAV（见 darkside-commodity.md）
         └── 困境公司 → 概率加权 DCF + 期权法（见 darkside-commodity.md）
```

# 第二部分：公允市盈率法（Fair P/E）——Schmidlin 六步构建法

## 来源

Schmidlin《估值的艺术》（The Art of Company Valuation）第 8.2.1 节。

**核心定位**：周期股首选方法，也是所有有稳定利润公司的有效估值工具。**周期股不用当期 EPS 做公允 PE，必须使用正常化 EPS。**

**《估值的艺术》核心哲学**：

> "公司估值更多的是一门艺术而非一门精确的科学。"
>
> "We do not claim that this model will provide the exact fair value of a share. But we do claim that it provides a systematic and transparent framework for thinking about valuation."

## 公式总览

$$每股公允价值 = 每股预期利润（正常化EPS） \times 公允市盈率$$

$$公允市盈率 = 基准PE + 财务稳定性溢价 + 市场地位溢价 \times 盈利能力倍数 + 增长溢价 + 独特性溢价$$

六步构建，每步独立评分，最终加总得到公允 PE。

---

## 第一步：基准市盈率（7–8x）

基准 PE = 7–8 倍（历史实证下限）。

**解释**：8 倍 PE 对应 12.5% 初始收益率（1/PE），接近股市长期必要回报率。标普 500 历史均值约 16.4 倍，7–8 倍是真实的估值地板——2013 年底不到 0.5% 的标普 500 成分股 PE 低于 8 倍。

**Schmidlin 原文**："If everything else were normal, a stock should trade at 7–8x earnings. Anything below that implies something is fundamentally wrong with the company."

---

## 第二步：财务稳定性溢价（0–2x）

| 指标 | 健康（+2分） | 一般（+1分） | 危险（+0分） |
|------|------------|------------|------------|
| 权益比率 | >40% | 20–40% | <20% |
| 净负债/EBITDA | <2x | 2–4x | >4x |
| 动态资本负债比率 | <3年 | 3–6年 | >6年 |
| 利息保障倍数 | >10x | 4–10x | <4x |

**评分规则**：
- 多数指标健康 → +2x
- 多数指标一般 → +1x
- 多数指标危险 → +0x
- 指标矛盾时（如低负债但低保障倍数），按最低指标定级

---

## 第三步：市场地位溢价（0–3x）

用**波特五力**评分（每项 0–5 分，总分 0–25 分）：

| 波特五力 | 0分 | 5分 |
|---------|-----|-----|
| 竞争强度 | 完全竞争 | 垄断 |
| 新进入者威胁 | 无进入门槛 | 进入门槛极高 |
| 供应商议价能力 | 供应商主导 | 公司主导 |
| 客户议价能力 | 客户主导 | 公司主导 |
| 替代品威胁 | 大量替代品 | 无替代品 |

| 总分 | 市场地位 | PE溢价 |
|-----|---------|--------|
| 20–25 | 垄断/寡头 | 3x |
| 15–19 | 强 | 2x |
| 10–14 | 中等 | 1x |
| <10 | 竞争激烈 | 0x |

**交叉验证**：将波特分与 EBIT 利润率对照——利润率 >30% 应对应波特分 22–25；5–10% 对应 8–12。两者严重不匹配时重新审视打分。

---

## 第四步：盈利能力倍数（0.5–3x）

使用**非杠杆 ROE**，排除财务杠杆人为放大：

$$最低权益比率 \approx \frac{资本支出}{经营性现金流}（取5年均值）$$

$$非杠杆ROE = \frac{净利润}{总资产 \times 最低权益比率}$$

| 非杠杆 ROE | 倍数 |
|-----------|------|
| >25% | 3.0x |
| 15–25% | 2.5x |
| 10–15% | 2.0x |
| 5–10% | 1.5x |
| <5% | 0.5x |

**综合溢价 = 第三步 PE 溢价 × 第四步倍数**

示例：市场地位 2x，非杠杆 ROE 30%（3.0x 倍率）→ 综合溢价 = 2 × 3.0 = 6x

---

## 第五步：增长溢价（0–6x）

### 核心原则（Schmidlin 强调的三条硬规则）

1. **用利润增长率，而非收入增长率**
2. **用完全稀释的每股利润增长率**
3. **至少 5 年时间框架**

| 预期 EPS 年增速 | PE 溢价 |
|---------------|--------|
| >25% | 6x |
| 15–25% | 5x |
| 10–15% | 4x |
| 5–10% | 3x |
| 3–5% | 1–2x |
| 0–3% | 0–1x |
| 负增长 | 0x |

### ⚠️ 增长率填写三条硬规则

1. **填入的是正常化后的中长期 EPS 增速（5 年维度），不是近期高景气下的增速。** 周期股在景气高峰期的利润增速可能达到 30%+，但这不是可持续增速，不能直接填入。

2. **区分利润增速与收入增速。** 利润率改善期利润增速会高于收入增速，但利润率改善是一次性的，不可永续——估值中应只用可持续的利润增速。

3. **重资产公司需扣除资本开支对增长的消耗。** 如果一家公司需要每投入 2 元资本才能新增 1 元利润，其"增长"的质量远低于轻资产公司，应在增长溢价上打折。

### 增长率验证

用反向 DCF 推算市场当前价格隐含的增长率——若隐含增长率远低于合理预期，股票可能被低估。

---

## 第六步：独特性因素（-2–+2x）

| 因素 | 调整 |
|------|------|
| 净现金 > 市值 20% | +1x（或直接加回现金） |
| 并购/私有化预期 | +1x |
| 长期合同锁定利润 | +1x |
| 政治/监管/诉讼特别风险 | -1–-2x |
| 中国 A 股：国有背景保护 | 酌情 +0.5–1x |

---

## 完整示例（Schmidlin 原例：劲量公司）

```
基准 PE：              8x
财务稳定性：           +1x（权益比率 30.7%，净负债率 87.6%，一般）
市场地位 × 盈利能力： 波特分 15 → 2x；非杠杆 ROE 30% → 3.0x → 综合 2×3.0 = 6x
增长溢价：             +1x（预期 4–5% 增长）
独特性：               0x
──────────────────────────
公允 PE = 8 + 1 + 6 + 1 + 0 = 16x
```

---

## 周期股正常化 EPS 的计算

Fair P/E 应用于周期股时，**必须使用正常化 EPS，绝对不能用当期 EPS**。

### 方法一：历史均值法

$$正常化EPS = 近5\sim10年（含完整高峰和低谷）的EPS均值$$

### 方法二：产能 × 价差法

$$正常化EPS = \frac{正常年景产能 \times 历史均价差 \times (1 - 正常税率)}{总股本}$$

### 方法三：Shiller PE

用 10 年均值 EPS 计算周期调整 PE。优点是平滑周期，缺点是不捕捉结构变化。

### 为什么当期 EPS 不能用

周期底部当期 EPS 是价格暴跌的结果，不代表正常盈利能力。用底部 EPS 算估值，等于假设行业永远不会复苏——逻辑硬伤。

---

## Schmidlin 的 CAPM 替代方案

当 Beta 历史数据不可靠时（刚上市、业务重大变化），可用 Fair P/E 倒推股权成本：

$$k_e = R_f + \frac{1}{Fair\ P/E}$$

Fair P/E = 16x 时，隐含风险溢价 = 1/16 = 6.25%，加上无风险利率即为股权成本。

**适用场景**：Beta 无法可靠估计时的备选方案。不能替代 CAPM 作为主方法，但可作为交叉验证。

---

## Fair P/E 与 DCF 的关系

Fair P/E 的结果应与 DCF 结果交叉验证。如果两种方法结论差异 >30%，必须分析原因，不能取平均了事。

- Fair P/E 更直观，适合快速估值和同行比较
- DCF 更严谨，适合详细建模和敏感性分析
- **周期股：Fair P/E 为主（权重 50%+），DCF 为辅（权重 <20%）**
- **成熟公司：DCF 为主（权重 60%），Fair P/E 为辅（权重 40%）**

# 第三部分：EPV（盈利能力价值）

## 来源
Bruce Greenwald《价值投资：从格雷厄姆到巴菲特》；达摩达兰将其整合进估值框架。

## 核心思想

**不假设任何增长，只评估"公司以当前规模、当前盈利能力永续运营"的价值。**

$$EPV = \frac{NOPAT_{normalized}}{WACC}$$

其中：
- NOPAT_normalized = 正常化营业利润 × (1 - 税率)
- 正常化意味着：剔除周期性波动、一次性项目、临时因素
- WACC 反映公司当前的风险水平

## 维持性资本开支的处理

$$NOPAT_{normalized} = EBIT_{normalized} \times (1 - t) - Maintenance\ CapEx$$

- 维持性 CapEx ≈ D&A × 1.0–1.1（参考行业数据修正）
- 扩张性 CapEx **不**计入 EPV（因为 EPV 假设零增长）
- 如果公司 D&A 与实际维护支出差距大，需要单独估算

## EPV 的三种核心解读

**解读1：安全边际检验**

| 比较结果 | 含义 |
|---------|------|
| EPV > 市值 | 即使公司零增长，当前价格也低估——安全边际极强 |
| EPV ≈ 市值 | 市场定价反映了当前盈利能力，不含增长溢价 |
| EPV < 市值 | 市场价格包含了对未来增长的预期——需验证增长假设 |

**解读2：增长价值分解**

$$Value\ of\ Growth = DCF\ Value - EPV$$

- DCF Value > EPV → 市场为增长支付了溢价
- DCF Value ≈ EPV → 增长假设几乎不影响估值（可能是成熟消费品牌）
- DCF Value < EPV → 模型内部矛盾（DCF 假设的增长摧毁了价值——再投资率太高）

**解读3：护城河检验**

| 比较结果 | 含义 |
|---------|------|
| EPV > 重置成本 | 公司有护城河（现有资产产生的回报 > 新建同样资产的回报） |
| EPV ≈ 重置成本 | 无明显护城河（超额回报 ≈ 0） |
| EPV < 重置成本 | 资产效率低下，或资产已过时 |

**重置成本**：在当前价格下重建相同规模经营能力所需的成本。

## EPV 的局限

1. **只回答"值不值得"**：不告诉你增长是否合理，只告诉你"不增长值多少"
2. **正常化有主观性**：什么是"正常"利润？不同分析师可能有不同答案
3. **不适用于快速变化的公司**：如果公司当前状态不代表未来（如正在重组），EPV 参考价值有限

## EPV 与巴菲特"盈利能力价值"的关系

**《巴菲特致股东的信》中的核心概念**：

> "The value of a business is determined not by today's earnings, but by its earning power — what it can reliably earn year after year."
>
> "企业的价值不是由今天的盈利决定的，而是由盈利能力——它年复一年能可靠赚取的利润——决定的。"

巴菲特的"盈利能力"与 Greenwald 的 EPV 本质相同。巴菲特用"看穿式收益"（Look-through Earnings）来正常化利润，Greenwald 用 NOPAT_normalized。

# 第四部分：APV（调整后现值法）

## 来源
Myers（1974）；《Investment Valuation》第3版 Ch.15

## 核心思想

将公司价值分解为两部分：

$$Value = Value_{Unlevered} + Value_{Tax\ Shield} - Value\ of\ Distress\ Costs$$

1. **Value_Unlevered**：假设公司全股权融资时的价值 = FCFF / k_u（k_u = 无杠杆股权成本）
2. **Value_Tax_Shield**：利息税盾的现值 = Σ(t × Interest_t / (1+k_d)^t)
3. **Value_of_Distress_Costs**：高杠杆导致的预期财务困境成本（可选）

## APV vs WACC DCF

| 维度 | APV | WACC DCF |
|------|-----|---------|
| 债务处理 | 分离处理（税盾单独估值） | 合并处理（通过 WACC 中的 (1-t)） |
| 适用杠杆率 | 随时间大幅变化 | 相对稳定 |
| 计算复杂度 | 略高 | 略低 |
| 透明度 | 更高（能看到税盾价值多少） | 较低（隐藏在 WACC 中） |
| 直觉性 | 更强（价值 = 基础价值 + 税盾） | 较弱 |

## APV 适用场景

1. **LBO（杠杆收购）**：杠杆率从 30% 飙升到 70%+，WACC 假设失效
2. **项目融资**：每个项目有独立的资本结构
3. **财务重组**：公司正在经历债务结构调整
4. **资本结构变化明显的公司**：如回购计划、债务到期再融资

## APV 操作步骤

1. 用无杠杆 Beta 计算 k_u（无杠杆股权成本）：
   $$k_u = R_f + \beta_u \times ERP$$

2. 用 k_u 折现 FCFF → Value_Unlevered

3. 预测每年税盾 = t × k_d × Debt_t

4. 用 k_d 折现税盾 → Value_Tax_Shield

5. 总价值 = Value_Unlevered + Value_Tax_Shield - Distress_Costs

6. 净债务 → 股权价值

# 第五部分：CAPM 批评与替代方案

## CAPM 的已知缺陷

**《估值的艺术》第7章的系统性批评**：

| 缺陷 | 具体问题 |
|------|---------|
| 单因素局限 | 只用市场风险一个因子，无法解释规模、价值、动量等异象 |
| Beta 不稳定 | 同一公司不同时期回归 Beta 差异大 |
| 市场组合假设 | CAPM 假设投资者持有"市场组合"，实际上没人做到 |
| 正态分布假设 | 收益率分布有肥尾，CAPM 的方差度量低估极端风险 |
| 前视 vs 后视 | Beta 基于历史数据，但估值需要未来风险度量 |

## 替代方案1：Fama-French 三因子模型

$$k_e = R_f + \beta_M \times ERP + \beta_{SMB} \times Size\ Premium + \beta_{HML} \times Value\ Premium$$

- **SMB（Small Minus Big）**：小公司长期跑赢大公司
- **HML（High Minus Low）**：价值股（高账面市值比）长期跑赢成长股

**优势**：更好地解释历史收益截面，R² 显著高于 CAPM
**劣势**：因子溢价未来是否持续不确定；三个 Beta 三个误差来源

## 替代方案2：套利定价理论（APT）

$$k_e = R_f + \sum_{i=1}^{n} \beta_i \times Risk\ Premium_i$$

- 不指定具体因子（不像 Fama-French 预设规模和价值因子）
- 因子通过统计方法（因子分析）从数据中提取
- 理论上更优，但实践中因子识别困难

## 替代方案3：基本面风险溢价

不用市场 Beta，而是用公司基本面特征直接估算股权风险溢价：

| 风险维度 | 指标 | 调整 |
|---------|------|------|
| 经营风险 | 收入波动率、经营杠杆（固定成本占比） | 波动越大 → ERP 加成 |
| 财务风险 | D/E 比率、利息保障倍数 | 杠杆越高 → ERP 加成 |
| 规模风险 | 市值、收入规模 | 规模越小 → ERP 加成 |
| 行业风险 | 行业周期性、监管风险 | 周期性/高风险行业 → ERP 加成 |

**适用场景**：非上市公司（无市场 Beta）、历史数据不足的新公司、跨国公司（无合适的指数做回归）

## Damodaran 的立场

**《Investment Valuation》第3版第4章**：

> "I use the CAPM not because I believe it is the perfect model, but because it is the most transparent and the easiest to replicate. If you use a multi-factor model, you add complexity without necessarily adding precision — you now have three betas to estimate, each with its own error."
>
> "我使用 CAPM 不是因为我相信它是完美模型，而是因为它最透明、最容易复制。如果你用多因子模型，你增加了复杂性但不一定增加了精确度——你现在要估计三个 Beta，每个都有自己的误差。"

**实务建议**：
- 以 CAPM 为基准方法
- 用替代方案做敏感性分析
- 如果替代方法给出的 k_e 与 CAPM 差异 >2pp，分析差异来源并决定哪个更合理

# 第六部分：相对估值系统框架

## 核心原则

**《Investment Valuation》第3版第18章 +《学会估值》第7章**：

> "Relative valuation is not an alternative to DCF — it is a complement. The question is not 'Is the P/E too high or too low?' but 'Is the P/E justified by fundamentals?'"
>
> "相对估值不是 DCF 的替代——它是补充。问题不是'P/E 太高还是太低？'而是'P/E 是否有基本面支撑？'"

## 倍数的分类

| 倍数类型 | 公式 | 驱动因素 | 适用场景 |
|---------|------|---------|---------|
| P/E | 股价 / 每股收益 | 增长率、风险、 payout | 盈利稳定的公司 |
| Forward P/E | 股价 / 预期 EPS | 同上（更前瞻） | 有可靠盈利预测 |
| PEG | P/E / 增长率 | 增长效率 | 高成长公司 |
| P/B | 股价 / 每股净资产 | ROE、增长、风险 | 资产密集/金融公司 |
| P/S | 股价 / 每股收入 | 利润率、增长 | 亏损但收入增长的公司 |
| EV/EBITDA | EV / EBITDA | ROIC、增长、再投资 | 跨行业比较 |
| EV/Revenue | EV / 收入 | 利润率、增长 | 亏损公司 |
| EV/FCF | EV / 自由现金流 | 现金质量 | 现金流为正的公司 |

## 倍数与基本面的关系

### P/E 的基本面驱动

$$\frac{P}{E} = \frac{Payout\ Ratio \times (1+g)}{k_e - g}$$

**推论**：
- 高增长 → 高 P/E（合理）
- 低风险（低 k_e）→ 高 P/E（合理）
- 高 payout → 高 P/E（有限制——payout 不可能 >100%）

**公允 P/E 的估算（Gordon 公式法）**：
1. 确定公司的可持续增长率 g = ROE × (1 - Payout Ratio)
2. 确定 k_e
3. 确定 payout ratio
4. 代入公式 → 得到"基本面支持的 P/E"

**⚠️ g 的永续增长率 vs 可持续增长率——极易混淆的陷阱**：

> **P/B 公式和公允 P/E 公式中的 g 是永续增长率（2–4%），不是可持续增长率（ROE × 留存率）。**
>
> 可持续增长率是**短期**参数（5–10 年维度），代表公司当前 ROE 和再投资能力支撑的增长速度。永续增长率是 DCF 终值/公允倍数公式中的**长期极限**，理论上限为名义 GDP 增长率（通常 2–4%）。
>
> **如果将 8% 的可持续增长率代入 P/B 公式，等同于假设公司永远以 8% 增长——在数学上隐含了规模无限扩张，不符合 Damodaran 框架的本意。**
>
> 这是估值中最常见的系统性错误之一，会导致周期股/高成长股的 P/B 和公允 PE 被严重高估。

**与同行比较时**：
- 不能简单比较 P/E 数字——必须控制增长率和风险差异
- "这家公司 P/E 30×，行业平均 20×" → 可能这家公司增长率更高、风险更低，30× 是合理的

### PEG 的陷阱

$$PEG = \frac{P/E}{g}$$

**问题**：
- PEG 假设 P/E 与 g 是线性关系——但公式告诉我们它是非线性的
- g 的定义不一致（历史 g？预测 g？3 年 g？5 年 g？）
- 没有考虑风险差异
- g = 2% 和 g = 20% 的 PEG 含义完全不同

**正确用法**：
- 仅在同行业、同风险等级的公司间比较
- 作为粗筛工具，不能作为最终估值依据

### P/B 与 ROE 的关系（金融公司）

$$\frac{P}{B} = \frac{ROE - g}{k_e - g}$$

⚠️ **此公式中的 g 是永续增长率（2–4%），不是可持续增长率（ROE × 留存率）。** 详见上方警示。

详见 darkside-financials.md。

### EV/EBITDA 的优势与局限

**优势**：
- 不受资本结构和税率影响（EBITDA 是税息折旧前利润）
- 适合跨行业比较
- 适合跨国比较（税率不同不影响比较）

**局限**：
- 忽略了资本密集度的差异（两家 EBITDA 相同的公司，如果 CapEx 差异大，EV/EBITDA 相同不代表估值相同）
- EBITDA 不是自由现金流——它没有扣除 CapEx 和营运资本变化
- 折旧摊销是真实的经济成本，不应该被忽略

**修正版**：EV/EBITDA 应配合 EV/FCF 一起看，两个倍数交叉验证。

## 相对估值的正确流程

1. **选择可比公司**：
   - 同行业（首选）
   - 相似规模
   - 相似增长阶段
   - 相似盈利能力
   - 同一市场（如 A 股 vs 港股 vs 美股，流动性差异大）

2. **标准化倍数**：
   - 用 Forward 倍数而非 Trailing 倍数
   - 剔除一次性项目对分母的影响
   - 注意会计准则差异（IFRS vs US GAAP vs 中国 CAS）

3. **基本面调整**：
   - 将倍数差异归因于增长率、风险、利润率差异
   - "这家公司倍数低于同行，是因为 ROE 更低还是因为增长更慢？"
   - 用截面回归控制基本面差异

4. **结论**：
   - 如果基本面调整后仍显著偏离同行 → 可能有估值异常
   - 估值异常不等于错误——可能是市场定价低效，也可能是你的假设有问题

# 第七部分：替代成本法与清算价值

## 替代成本法（Replacement Cost Method）

### 核心思想

公司的价值不应超过**重建相同经营能力所需的成本**。

$$Value \leq Replacement\ Cost$$

### 估算方法

| 资产类型 | 替代成本估算 |
|---------|------------|
| 有形资产 | 当前市场购置价格（土地、建筑、设备） |
| 品牌/无形资产 | 重建品牌所需的营销投入（历史营销支出的资本化价值） |
| 技术/专利 | 类似技术的授权费用现值 |
| 客户关系 | 获客成本 × 客户数 × 留存率调整 |
| 组织能力 | 重建团队/流程的时间和成本（很难量化） |

### 应用

- **Tobin's Q** = 市值 / 重置成本。Q > 1 说明公司创造了超额价值；Q < 1 说明公司资产效率低下
- **估值上限检验**：如果 DCF 值 > 重置成本 × 2，需要极强的护城河证据
- **并购定价**：收购方不会支付超过重置成本太多的溢价（否则不如自己建）

### 局限

- 替代成本很难精确计算（尤其是无形资产）
- 有些资产不可替代（如独特地理位置、历史品牌）
- 不反映未来增长价值

| 公司类型 | 最低安全边际 | 说明 |
|---------|------------|------|
| 稳定可预测公司 | 20–30% | 消费品、公用事业等 |

# 第八部分（续）：金融（证券/资管）专项

## 来源
Damodaran《Dark Side of Valuation》第3版 Ch.18（Financial Service Firms）；darkside-financials.md；2026年券商估值实践总结。

## 为什么证券/资管需要独立分节

券商的收入结构（经纪/自营/投行/资管）与银行（存贷利差）和保险（保费/赔付）完全不同。核心差异：

| 维度 | 银行 | 保险 | 证券/资管 |
|------|------|------|----------|
| 收入波动性 | 低（利差稳定） | 低（保费稳定） | **极高**（自营随市波动） |
| ROE波动范围 | 8-15% | 5-12% | **-10%~25%** |
| 债务性质 | 存款（经营性） | 准备金（经营性） | 多数为融资性 |
| 适用P/B-ROE | 较好 | 尚可 | **差**（ROE不满足稳定假设） |

**核心结论：P/B-ROE 简化公式对券商基本不适用，FCFE DCF 为主方法。**

## 典型倍数范围

| 指标 | 典型范围 | 备注 |
|------|---------|------|
| P/B  | 0.7~2.0x | 高度周期依赖；熊市底部常破净 |
| P/E  | 10~25x  | 周期顶部P/E低（利润虚高），底部极高或负，慎用 |
| ROE  | 均值5~9%，牛市可达20%+ | 必须正规化，单年数字无意义 |

## P/B-ROE 公式对券商为何失效

$$\frac{P}{B} = \frac{ROE - g}{k_e - g}$$

此公式成立的前提（全部被券商违反）：

| 前提 | 要求 | 券商实际情况 |
|------|------|------------|
| ROE稳定 | 永续期内保持不变 | 2008-2025年东吴ROE从1.75%到21.74%，方差极大 |
| 分红率稳定 | 保持固定比例 | 牛市分红率提升、熊市降低或取消 |
| 再投资ROE=当前ROE | 新增投资回报等于存量 | 佣金下行周期中，新增资本回报低于历史均值 |

**结论：P/B-ROE 可作辅助交叉验证，不能作主估值方法。**

## ROE 正规化方法

简单均值受牛市年份严重扭曲。推荐方法：

| 方法 | 操作 | 适用场景 |
|------|------|---------|
| **指数加权均值（首选）** | 半衰期5年，保留全历史但近年权重更高 | 有10+年数据时使用 |
| 周期中位数 | 去除牛市/熊市极值后取中位数 | 牛熊特征明显的市场 |
| 全面摊薄ROE | 净利润 / 期末BPS | 作为下限参考 |

同时必须列出悲观/基准/乐观三个情景，不能只给一个"正常化ROE"。

## 首选方法：FCFE DCF

**为什么 FCFE DCF 适合券商**：
- 不依赖"稳定ROE"假设——可分阶段预测不同ROE
- 不依赖"永续增长"假设——终值用退出P/B，只占总价值30-50%
- 可以显式建模佣金下行、自营波动等周期性因素
- DDM 对分红率假设极度敏感，为辅助参考

**FCFE DCF 对券商的操作要点**：
1. 5-10年分阶段预测EPS（反映ROE从当前向趋势值回归）
2. 分红率随周期调整（牛市可提高、熊市保守）
3. 终值用退出P/B（基于趋势ROE和趋势分红率计算）
4. 敏感性分析必须覆盖：Ke（Beta选择）、终值ROE、永续g

**不适合的方法**：
- EV/EBITDA（券商债务多为融资性，不是经营性的）
- DDM作为主方法（分红率波动太大）
- P/B-ROE作为主方法（见上方分析）

## Ke 参数校验（券商必做）

用历史PB反推隐含Ke，与CAPM估算值交叉验证：

$$k_{e,\text{implied}} = g + \frac{ROE - g}{PB_{\text{historical}}}$$

**校验规则**：
- 如果CAPM Ke与历史隐含Ke均值相差超过200bp，需分析原因
- 差距可能来自：MRP假设过高/低、Beta选取偏差、账外资产（牌照价值）
- 校验目的是发现假设错误，不是把Ke调低到"复现当前股价"

**券商Beta参考**（Damodaran 2026年1月，32家美国券商）：
- Levered Beta: 1.356
- Unlevered Beta: 1.171
- 框架参考范围: 1.2-1.8

**中国券商Ke参数建议**（基于Damodaran 2026年4月数据）：
- 无风险利率: 中国10年国债收益率（2026-05为1.77%）
- ERP: 中国总ERP = 5.69%（CDS法）或 5.80%（评级法），含CRP约0.9-1.1%
- Beta: 1.17（Unlevered）或 1.36（Levered），按实际杠杆调整
- Ke = Rf + Beta × ERP ≈ 8.4%~9.6%
